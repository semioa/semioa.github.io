<div class="_wbtext_hwnxp_14">
	今天我来讲讲，科研人员为什么一定要有生活和副业这件事。<br/>
我从本科毕业到博士三年级的这五年，基本上科研就是我的一切。沉迷科研的时候，衣服不买了，妆不化了，外型不在意了，活动不参加了，朋友也不交了，所有时间都在想着前进，优化，前进，优化。收获是很明显的，我博三的时候就有一篇顶刊publish，一篇顶刊minor revision，还有一篇二年级时候花了三个月写的论文半年不到就发表了。<br/>
但是代价也很惨重。三年级结束的时候我在科研上遇到了严重的存在主义危机，跟老板干架，休学邮件写好了，准备休息一个月就退学去Whistler打零工，打算所有的一切都不要了从头开始。诱因其实也很trivial，一个我写了三年做了一百多个实验的paper突然重复不出来了。我承受不了这个打击。<br/>
后来我觉得，既然我已经什么都不想要了，不如把博士当成玩一样地读，拿到学位去个雪山旁边的teaching school，收入总归比在Whistler打零工多一些。那时候我认识了现在的老公，在学妹带领下入坑Kpop，开始大量的滑雪和跳舞。老公也和我说，他爱我跟我的CV什么样一点关系都没有，如果有一天我要辞职去当舞蹈老师，他也坚决支持。<br/>
我那时候真的没想到这两件事后来撑住了我。它们怎么撑住我的？它们让我意识到，我的人生并不只有科研一件事。科研就算毁掉，我还可以滑雪，还可以跳舞。我的雪滑的不怎么样，舞也跳的不怎么样，在这两件事上我的天赋远不如科研和做题。<br/>
但是它们的存在，使得我的存在不再依附于一件难度极高、随机性极大且回报非常任意的事情。在科研之外，我仍然可以从别的事情里获得乐趣、保障身心健康，并继续进步。这个buffer非常重要，不亚于父母亲和我说不想读了回家躺平完全可以养我，也不亚于丈夫说支持我的事业是因为我喜欢，如果我有一天不喜欢了就干别的喜欢的事情，一点不影响他爱我。<br/>
我最近经常在社交媒体上刷到一些奇谈怪论，比如为了一个帽子可以卖房去打通关系。我震撼也悲伤，人竟然可以被异化到这种程度。现代社会的生产力下，很多所谓的生死存亡的危机，都来自于人的幻想。拥有一个普通家庭和健全的身体，人就不可能会饿死的，那么很多所谓“不这么做就会死”都大抵是一种妄念。我也时常会心灵重感冒，被妄念支配，但人都会有被妄念支配的时候，不代表就应当被妄念支配。<br/>
你一直有选择的，真的一直有。

	目前公开的志愿军战后精神病患者和PTSD案例超过1000人，<br/>
沙岭子医院的护士李文茂回忆：<br/>
江兴汉，对枪极度钟爱以至于病态，见到什么东西都要掰成或组装成枪并随身佩戴，连拿到两个青霉素药瓶都要摆成手枪状，医护一但想收走他的“枪”就会变得暴躁。每天的活动就是拿着“枪”瞄准所有周围的人和物。<br/>
马光耀，易怒，嗜咬，咬一切东西和靠近的人，没得咬就咬自己，甚至咬断了手指，医护怕他自伤，不得已只能每天把他捆起来。<br/>
李海荣，抗拒穿衣服，春夏秋冬都是裸体状态（直到改开后有媒体采访，摄像头拍到老兵还是光着身子，拒绝穿衣服）。<br/>
赵桐风，病发就大吼并冲锋陷阵，平时缩在角落里自己画图，说是要设计威力无穷的巨炮（赵是侦查连连长，曾荣获志愿军“特级英雄”功勋，回国后54年突然发病）。<br/>
李裕，高炮营长，上甘岭一级战斗英雄，受到过毛主席接见。退伍后发病，每天妄想会有空袭，把任何声音都当成敌机过来轰炸。 ――2010年沙岭子最后一名志愿军精神病患张洪恩病逝。<br/>
湖州市第三人民医院的护士俞静回忆：<br/>
谭才生，每天凌晨2点固定起来劳动，打扫擦洗周围的家具杂物，怎么都劝不住。护士想了个办法，严厉告诉他“遵守纪律，不许凌晨两点起床，这是命令！”才能消停。<br/>
40军战士洪潮林，第四次战役中被俘，1953年释放回国后精神失常，在湖州疗养时时而正常，时而发病，谩骂和摔打，随意泼洒排泄物，1959年去世。<br/>
马玉堂，20军的副班长，入朝时一直作战英勇，51年一天阵地遭到敌人猛烈炮击时突然精神崩溃，并多次试图自杀，1955年被强制退役送到湖州三院治疗，每天都是缩在角落里，拒绝见人，拒绝参与一切活动。<br/>
2010年湖州三院仅剩2名还在世的志愿军病患，其中一位胡久昌已经病危，在他还没病危前有极强的暴力倾向，攻击一切靠近的人。<br/>
另一位蒋震娟于16岁就参加志愿军，然而1年后就精神失常，19岁进入湖州三院接受治疗。直到2010年，俞静带着媒体询问蒋震娟“你几岁了？”，75岁的老兵仍然回答“17岁！”她的内心和那些战友们一样，从来没有从战场回来。

	【3】<a class="_cut_ygi5b_97 _name_ygi5b_120 _default_fvu9w_3 _cut_ygi5b_97 _name_ygi5b_120" href="https://weibo.com/u/1726294583">荐见</a>

<p>
	讲个故事。前几天，人社部、财政部联合发了个通知：支持大学毕业生回炉去读技校。支持技工院校招收离校未就业高校毕业生。你没听错。拿着本科、大专文凭的人，可以再回去学一门手艺。这背后究竟意味着什么？我们今天来聊聊，这个故事一定值得你听到最后。<br/>
<br/>
首先是个数字:1270万人。今年毕业。历史新高。校招岗位560万。两个人抢一个坑。这是2026年的中国就业市场。<br/>
<br/>
这“回炉”可不是来虚的。北京已经动起来了。今年北京推出6个全日制大学生技师班，专业涵盖智能制造、集成电路、新能源汽车。学制2年，1年在校、1年企业实习。毕业拿双证，推荐就业。其实从去年开始，广东、浙江、山东也都已经开始试点了。这次人社部发文，是把地方经验全国化了。<br/>
<br/>
最直接的动因还是来自数据，来自市场反馈。技工院校就业率连续多年稳定在97%以上。本科毕业生呢？很多人送外卖了。把暂缓就业、灵活就业都算进去，大概80%。<br/>
<br/>
一边是1270万毕业生挤破头抢文职岗，月薪四五千还不稳定。一边是智能制造、新能源、集成电路行业开8000+月薪，招不到能上手的技术人才。<br/>
<br/>
人对不上号。所以有了"回炉"。<br/>
<br/>
青岛技师学院2025年大学生技师班招生超240人。徐州工程机械技师学院有30多个本科生"回炉"学智能制造。政策给的实惠也实打实――学费全免，每月发生活补贴，考证费用报销。据说有个学体育的年轻人转行学电气自动化，没毕业就被大企业抢走了。企业说：我不为一张空文凭买单，但我愿意为拿来就用付钱。<br/>
<br/>
那么问题来了：如果大学四年烧出来的是废铁，现在重新扔回炉子，烧成一把趁手的扳手。那大学四年算什么？时间成本、家庭投入、社会资源，就这么一笔勾销了？更深层的问题是：为什么会走到这一步？一个人读了四年书，为什么毕业发现自己什么都不会？是学生不努力？是教的东西本身就有问题？<br/>
<br/>
人们很容易就因此去批判传统高等教育模式的垮掉，批判教育和市场和现实的脱节，对吧？但你想过没有，这还有一个细思极恐的问题是：事实上，我们的教育，一直在跟着市场和现实跑啊。现在高校膨胀出的这些专业，那些被人弃若敝屣的专业，市场营销，国际贸易，包括新闻传播，哪个当年不是招生的热门，市场的香饽饽啊。张雪峰生前生后为什么会被这么多人奉若神明？不就是因为看上去他能指出专业和现实间的信息错配，用一种极致实用主义的价值观，教人们在真实世界里先活下来嘛。<br/>
<br/>
可然后呢，真的就能活下来吗？当年扩招，根本上也是因为市场需要大学生啊，然后，高等教育毛入学率从30%涨到60%。大学生也快速从供不应求转向成为供远大于求。<br/>
<br/>
追着市场跑，永远追不上的。市场变太快了。还记得马斯克说的嘛，我们不是在走向奇点，我们已经活在奇点之中了。AI这两过山车刚刚到达顶点，即俯冲直下。刻舟求剑还有用吗？有机构预测，2025到2030年，人工智能将替代900多万个岗位。编辑、翻译、基础编程，这些岗位需求正在萎缩。<br/>
<br/>
今天你在技校学到的技能，明天可能就过时了。今天回炉的产线，明年可能就被全流程智能化了。你永远赶不上的啊！技能重不重要，重要啊，当年就不该让那么多人全部涌向高校。应该有计划地发展优质的职业教育。而同理，现在，通识教育，人文教育就不重要了吗？<br/>
<br/>
过去跟着市场跑，出了问题。现在继续跟着市场跑，就能解决问题吗？我们一直没有反思的恰恰是：教育的真正本质是什么？其实过去我们大力发展高等教育，并没有真正面对这个问题，并不是以“人”为出发点，恰恰是从当时的市场出发。当教育只剩下"和市场对接"这一个目标，我们会失去什么？<br/>
<br/>
今年《华尔街日报》有过一篇刷屏的文章《人工智能公司的几位高管是怎么向子女描述未来职业前景的？》，我推荐过。里面提到了AI时代最核心的能力是什么？恰恰是现在被很多人看不上的文科教育培养的，这篇文章称为一个人的“元认知能力”。因为现在和AI的交互，都是通过自然语言的意图表达。批判性思维、适应性、提供语境和挑战AI结果的能力，伦理判断力，就显得无比重要。<br/>
<br/>
你看，元认知能力不是教你怎么操作一台机器，而是教你在机器被淘汰之后，还能做什么。不是教你怎么完成一项任务，而是教你在任务本身发生变化时，怎么重新定义问题。只有真正掌握这种底层能力的人，才不会在技术的快速迁移和流动里，不被单一技能绑死。一句话：技术让技能变得廉价，但让人性变得珍贵。<br/>
<br/>
什么是真正的人才？是能理解技术逻辑、能看见人性需求、能在混乱中找到方向的人。教育不能只看眼前。大学的尽头不该是技校。不该是工具。大学的尽头是人，是自我――是你找到一件愿意投入全部身心的事情，无论它在哪里。<br/>
<br/>
有些东西可以回炉。但有些东西，一旦烧掉了，就再也回不来了。你认同吗？欢迎点亮右下角红心，留言区留下你的观点，我们一起聊聊。

	刷到一条红薯，有人认为穿越到战国，去秦国更好，理由是秦国能吃得饱。并且他给出了文献理由，《史记》：“（郑国）渠成……四万余顷，收亩皆一钟。”相对比，《管子》说“百亩之收，不过二十钟。”于是他认为，秦国的亩产达到了山东六国的五倍左右。<br/>
其实，灌溉确实是农作物收成的重要基础，但是保证灌溉并不能凭空突破生产力。所以同时代东西两面收成相差这么多，并不现实。那为什么两个文献的数字会相差五倍呢？又为什么正好是五倍呢？<br/>
巧了，商鞅改革的一项内容，就是改变秦国的亩制，从小亩变成了大亩制。而两者的比率恰好就是五倍左右，即大亩的一百亩，等于原小亩的五百亩。所以《管子》所说，应该就是“百（小）亩”，上限能收二十钟。而《史记》所说的，则是秦之大亩，每百亩，皆能收百钟。灌溉的作用是使秦之耕地普遍获得了好收成，而不是生产力大爆发，一亩能收六国的五倍。<br/>
文献检索和信息获取是这个时代特别容易的事情，如何综合思考和处理这些信息成了更可贵的能力。
