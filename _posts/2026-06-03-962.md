			人们经常听说吃荔枝上火，上火是个传统医学概念，那么为什么大量吃荔枝会引起不适？首先荔枝果肉（假种皮）含糖量极高，达20%，会引起口腔和咽喉高渗环境，造成黏膜脱水、干燥、破裂，并且容易滋生细菌，引起口腔溃疡，所以这是典型“上火”的感觉。但更重要的是，和同科作物桂圆不同，荔枝独特且危险的地方在于它含有次甘氨酸A（Hypoglycin A）和亚甲基环丙基甘氨酸（MCPG）。这两类毒素在人体空腹、体内糖原储备不足时大量摄入会引发较严重后果。它们通过抑制脂肪酸beta-氧化，阻断了肝脏的糖异生通路。当荔枝自身带来的高糖被身体代谢完毕或因胰岛素分泌而消耗后，身体由于毒素阻断了糖异生通路，无法通过非糖物质转化血糖，从而引发严重的急性低血糖和代谢紊乱，这就是荔枝病。 因此，民间所说的吃荔枝“上火”，其背后其实描述的是高糖引起的高渗促炎反应，以及毒素引起的急性代谢紊乱表现。

	【3】大家是不是很久没听到比尔・盖茨的消息了？『华尔街日报』今天发了篇特稿，披露了这位前世界首富的糟糕境遇。<br/>
<br/>
往年微软举办年度 CEO 峰会时，盖茨通常会在其华盛顿州的家中设宴招待列席的嘉宾。但就在今年峰会开幕前几周，盖茨团队收到微软方面的“建议”：希望他今年别再搞家宴，尽可能保持低调。<br/>
<br/>
祸不单行，刚刚落幕的伯克希尔・哈撒韦的年度股东大会中，作为克希尔・哈撒韦的董事，比尔盖茨竟然也破天荒缺席了这场活动。<br/>
<br/>
但这场缺席早有预兆，此前巴菲特已经辞去了盖茨基金会理事职务，并在 2024 年向《华尔街日报》明确表示，在他离世后，盖茨基金会将不会从他那里获得任何资金。<br/>
<br/>
两位相识数十年的老友之所以走到这个地步，原因就在于比尔・盖茨卷入了杰弗里・爱泼斯坦的丑闻。<br/>
<br/>
巴菲特向媒体特别进行了说明，自爱泼斯坦文件公开以来，他就再没和盖茨通过话。他想先观望一下，看看还能从这些文件中查出什么端倪，等到 6 月下旬再做进一步的决定。<br/>
<br/>
在爱泼斯坦事件发生前，比尔・盖茨可以说是一代人的科技偶像。他名下的盖茨基金会是全球最大的基金会之一，手握 890 亿美元捐赠基金，在全球健康与发展领域做出了开创性贡献。<br/>
<br/>
在 2019 年一项最受仰慕公众人物的民调中，盖茨高居全球榜首。<br/>
<br/>
这背后离不开一只强大的公关团队。<br/>
<br/>
盖茨的公关团队多达数十人，负责打理他个人及整个商业帝国的对外沟通。他们甚至专门备了一个按他体型定制的人体模型，用来为他每天的穿搭试装。<br/>
<br/>
据现任及前任员工透露，造型团队在一处外部办公楼内存放了大量中性色的圆领和 V 领毛衣、纽扣衬衫、休闲裤，以及多副备用的 Silver Lining Opticians Carbon 眼镜。<br/>
<br/>
一旦有公开活动，团队通常会搭配出三套方案，交由高层审核。这一切只为一个目的：呈现出一个如罗杰斯先生般从容且平易近人的形象。<br/>
<br/>
盖茨基金会及其私人办公室 Gates Ventures 的两支民调团队一直在密切追踪公众对盖茨的看法，指标涵盖好感度、可信度以及公众号召力，同时为他的社交媒体和个人博客“盖茨笔记”招揽粉丝。<br/>
<br/>
这个团队还试图通过赞助的方式，去修改与比尔・盖茨有关的纪录片。<br/>
<br/>
此前奈飞上线了纪录片《走进比尔：解码比尔・盖茨》，以极尽赞美且贴近私人的视角，展现了盖茨投身慈善的人生下半场。<br/>
<br/>
按照纪录片相关的制作规范，这部纪录片在初稿完成后被送到盖茨那进行审片。随后盖茨的公关团队向制作团队发送了一份长达九页的文件。<br/>
<br/>
文件中语气严厉地列出了多项具体反馈意见，甚至细致到要删除某几秒的某个表情。<br/>
<br/>
往来邮件中更显示，盖茨团队财大气粗地承诺道：“如果需要更多资金，我们可以全额甚至超额补偿，我们不希望成本成为不继续做下去的理由。”<br/>
<br/>
随着这部纪录片的成功播出，尝到甜头的盖茨团队开始筹备另一部奈飞剧集，该剧将他描述为一位科技远见者以及全球健康与气候慈善家，名为《未来探秘：比尔・盖茨引领前沿》。<br/>
<br/>
但所有的这些努力都因为爱泼斯坦而化为泡影。<br/>
<br/>
在今年 2 月的一场基金会全员大会上，盖茨向员工首度坦承，自己确实如爱泼斯坦邮件中所说，与两名俄罗斯女性有过婚外情，并严肃定性道：“此举与基金会的价值观背道而驰。”<br/>
<br/>
但他还是坚持认为：“我没做任何违法的事，也没看到任何违法的事。”<br/>
<br/>
多年来，盖茨及其副手一直声称，他与爱泼斯坦的关系纯粹出于慈善目的，且两人会面时没有女性在场。<br/>
<br/>
基金会内部沮丧与不满情绪开始蔓延，甚至有部分员工当场悲伤落泪。<br/>
<br/>
就在这次大会后不久，基金会遇到了第一件棘手的事：原定的盖茨印度之行，是否还有继续的必要？<br/>
<br/>
由于错估了形势，最终比尔盖茨还是踏上了去印度的旅程。他辗转孟买，最终抵达新德里。按原计划，他将在那里出席一场汇聚多国元首与科技领袖的 AI 峰会，并发表主题演讲。<br/>
<br/>
就在演讲前几天，盖茨的名字却从会议官网的搜索列表中消失了。<br/>
<br/>
印度官员单方面向当地媒体透露，鉴于盖茨卷入爱泼斯坦文件，他们正在重新评估对他的邀请。<br/>
<br/>
就在同一时期，盖茨基金会印度办事处在推特上发布公告，硬刚相关传闻：“比尔・盖茨正在参加 AI Impact Summit。他将按原计划发表主题演讲。”<br/>
<br/>
但就在盖茨原定登台的数小时前，基金会又紧急在推特上发布声明宣告：“经过慎重考虑，为确保各方焦点不偏离 AI 峰会的核心议题，盖茨先生将取消本次主题演讲。”<br/>
<br/>
紧接着，盖茨也取消了正在筹备中的南非之行，然后又缺席了能源界顶级盛会剑桥能源周。<br/>
<br/>
6 月 10 日，比尔・盖茨将因爱泼斯坦事件接受国会质询，盖茨团队聘请了前司法部官员、律师 John Moran 作为代理人。<br/>
<br/>
团队成功将这次自愿质询推迟了数周，并达成协议，确保盖茨在质询期间免于录像。<br/>
<br/>
而这场质询活动的结果，也许将成为比尔・盖茨此后事业的重要转折点。

	【6】以《历史的终结？》一文著称的美国政治学者福山接受德国《法兰克福汇报》采访时承认，如果中国继续保持目前的发展势态，那么他四十年前提出的有关中国的预言便是错误的。<br/>
<br/>
《法兰克福汇报》发表了对他的访谈。<a href="https://weibo.cn/sinaurl?u=https%3A%2F%2Fwww.faz.net%2Faktuell%2Ffeuilleton%2Fbuecher%2Fsachbuch%2Ffrancis-fukuyama-trumps-regierung-ist-die-korrupteste-in-der-us-geschichte-200886581.html" target="_blank"><img class="icon-link" src="/assets/images/193429/004.png" style="width:1em;height:1em;"/>网页链接</a>福山于1989年发表了《历史的终结？》并引起轰动。该文阐述了自由民主制度代表“人类意识形态演进的终点”这一观点，是“人类最终的执政形式”。福山认为，随着美国在冷战中的胜利，民主和市场经济将遍及全球。<br/>
<br/>
四十年后的今天，福山不得不承认他的预言没有出现，尤其特朗普前不久访问中国之后，福山将美国描述为“衰落的超级大国”。《法兰克福汇报》记者问道，美国的衰落是从什么时候开始的？<br/>
<br/>
福山答道：“1989年至2008年是历史上一个非同寻常的时期。美国在文化、经济和政治等各个领域都拥有前所未有的主导地位。2008年金融危机后，全球力量的平衡发生了转变。自特朗普上台以来，美国经历了真正的自我削弱。此前美国社会就已开始分裂，而特朗普上台更是加剧了两极分化的程度。美国在世界上应扮演怎样的角色？对此美国存在分歧。第二个任期上，特朗普明显降低了对中国的压力。中国人不禁要问：美国为什么要这样做？对中国而言，这无疑是一份厚礼。”<br/>
<br/>
记者问：中国的崛起以及威权统治模式与你的从前的论点是否自相矛盾？<br/>
<br/>
福山答道：“如果中国继续保持目前的态势――政治稳定、威权体制以及经济增长，那么我不得不承认，我对它提出的论点是错误的。虽然我曾提出过一个现代化的假设，认为随着中国中产阶级的崛起和民众教育水平的提高，人们对自由、法治以及向西方自由民主模式靠拢的诉求也会随之增长。但这种情况并未发生。不过，我目前还不认为中国将成为一个完美成功的模式。中国的治理模式难以输出，而且中国也存在弱点。房地产行业正经历着前所未有的低迷。总之，中国作为自由民主的替代模式，其成功程度目前尚不明朗。”

	2012年，我第一次到访德国，那会儿我着迷于铁幕东边的历史，到德国后，逢人就问东德的事，但我所在城市，是原属西德的汉堡，人们对东边根本没有兴趣，虽然当时的联邦总理默克尔就出身东德。<br/>
碰壁若干次后，我终于在一个饭局上，听一位德国前驻华记者说起，他有一位东德同学，经历蛮特别，可以和她聊聊。几周后，我从柏林出发，在开姆尼茨――东德时叫卡尔・马克思城――见到了安佳。我和安佳在一起呆了整整一天，听到了她不可思议的、像《窃听风暴》与《再见列宁》混合体的爱情故事，我把这个故事放在20世纪下半叶冷战与“历史的终结”的大背景下，写成了下面这篇特稿《另一个国度》，2012年发表在当时供职的《南方人物周刊》上。<br/>
在这篇特稿里，故事结束于2009年11月9日，柏林墙倒塌20周年的那个夜晚。但人们的生活仍在继续。我和安佳保持着通信，我也数次重返德国，2019年和2024年两次还重返开姆尼茨，与安佳相聚。<br/>
随着时间的推移，我意识到，这个故事有更多的元素，也有更复杂的纹理，而历史的推进，会一次又一次把我们以为的终点变成起点。我决定把这个故事做一些扩展。这个过程花的时间比我想象得长得多得多（我甚至在中间写完了一本《重走》），但总算完成了，结果就是我刚刚出版的非虚构作品《另一个国度：在20世纪废墟上的漫游》。<br/>
如果你读完了这篇特稿，也对我的同名新书感兴趣，不妨直接从第九章《寻找DDR》读起。当然，整本书其实讲述了同一个主题，借用这篇特稿里的一句话，到头来，人们得学会与过去相处，也许更重要的是，与自己相处。<br/>
