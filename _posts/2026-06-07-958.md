	我为什么不想去电影院看《给阿嬷的情书》？<br/>
一是因为很多人说很感人，我就不想看了。“找泪流”是我最不喜欢的一种电影类型，那种被精心设计好的情绪催熟，总觉得少了点对生活本身的尊重。<br/>
二是因为题材本身。大家听说过“番客婶”吗？<br/>
春节前，我们一家在潮汕旅游，爱偷听的小某书就给推了很多相关内容，其中包括一本关于“番客婶”的书――沈惠芬的 《社会性别与历史书写：20世纪上半叶福建移民家眷生命史研究》 。推文是这本书的介绍，我第一次知道“下南洋”背后“番客婶”这个名称。<br/>
但真正让我感到沉重的，是评论区。那里面有数百条评论，大部分IP来自广东和福建。留言大概是这样的：<br/>
“我太奶奶就是番客婶，太爷爷下南洋去了，头几年有书信和钱寄返，后来杳无音讯。太奶奶拉扯几个孩子，侍奉公婆，一生就在等。若干年后，等来的只是男人的死讯。”<br/>
或者是更扎心的：“男人回来了，带着在南洋娶的妻妾、儿女，浩浩荡荡回来祭祖。原配老丑衰残，望着衣锦还乡的男人。中间隔着几十年的时间汪洋大海，还有战争与断汇时期的艰难。”<br/>
我看得很难受。<br/>
我知道电影《给阿嬷的情书》票房很高，豆瓣9.1分，讲的是一个非常美好的故事：丈夫下南洋后杳无音讯，其实是另一位女性受丈夫所托，代他给家中阿嬷寄了一辈子的信和钱。这是人性里的光辉和善意，我相信现实中一定有这样的情义。<br/>
但我不想去影院看，是因为我清楚，文创作品的美化终究是有限度的。<br/>
在那段真实的历史里，这种“双向奔赴”的情义是稀缺品，更多的是单向度的绝望。所谓“番客婶”这个群体，本质上是传统宗族社会为了维持家庭稳定，而在牺牲掉的一代女性。<br/>
在沈惠芬教授的研究中，这些留守女性的生存状态被描述为一场“社会地位保卫战”。她们不仅要面对繁重的体力劳动，还要面对“害怕自家男人过洋变心，另组家庭”的恐惧。有相当一部分番客婶，穷尽一生等来的，要么是断供后饿死的命运，要么是丈夫在海外另娶的消息。甚至有番客婶留下遗言，死后不要和丈夫合葬，也不要将他的名字刻在自己的牌位上。<br/>
即便电影《给阿嬷的情书》里真的有情有义，有女女相助，但在这个故事架构中，男人依然是隐身的，而苦难却是实打实的。<br/>
去影院静静地、专注地看这么一个被精心包装过的美好故事，对我来说坐立难安。我不想坐在黑暗的，气味复杂的影院里，看着荧幕上那封美丽的情书流泪，去消费那段对无数“番客婶”而言过于残酷的历史。<br/>
我宁愿在那一条条真实评论的缝隙里，记住她们没有等来情书的、沉默的一生。

	【2】在最近的一场演讲中，大法官克拉伦斯・托马斯（Clarence Thomas）明确表达了他对进步主义（Progressivism）的看法，他表示，进步主义“是美国第一个公开反对《独立宣言》原则的主流政治运动――唯一可能的例外是内战前夕支持奴隶制的反动派”。他接着将进步主义与20世纪最严重的罪行联系在一起，声称它“导致了那些造成了世界有史以来最糟糕世纪的政府。斯大林、希特勒、墨索里尼都与进步主义的兴起交织在一起，且都反对作为我们《独立宣言》基石的自然权利。”<br/>
<br/>
众多研究进步主义时期的学者则辩称，托马斯提供的是一个被扭曲的运动版本。对于托马斯将斯大林、希特勒等独裁者与进步主义“交织在一起”的说法，历史学家认为这在逻辑和历史上都极其荒谬。<br/>
<br/>
历史事实正好相反：正是因为美国在20世纪初以及30年代罗斯福新政时期，通过进步主义式的改革（如建立社会保障体系、反垄断、保障工会权利等）缓解了社会矛盾，美国才得以在经济大萧条时期避免像德国、意大利那样滑向法西斯主义，或像俄国那样倒向共产主义极端体制。<br/>
<br/>
学者们强调，将旨在通过立法和行政监管来完善民主制度、改善民生的美国本土改革，与彻底消灭私有制、铲除个人自由、实行极权统治的欧洲及亚洲独裁政权混为一谈，是对20世纪政治历史的恶劣篡改。<br/>
<br/>
托马斯认为伍德罗・威尔逊等进步主义者用“政府赋予权利”取代了“上帝赋予的自然权利”。但学者们反驳道：进步主义者（包括老罗斯福、威尔逊等）并没有否定《独立宣言》中“人人平等”的原则，而是意识到，在工业化和垄断资本主义时代，如果缺乏政府对经济秩序的合理干预，弱势群体的“生命、自由和追求幸福的权利”根本无法得到实质性的保障。他们试图通过建立现代行政国家来践行《独立宣言》的承诺，而不是推翻它。<br/>
<br/>
许多法律学者和政治评论家指出，托马斯作为最高法院资深大法官，发表这样极具党派和意识形态色彩的演讲是极不妥当的：前保守派联邦法官迈克尔・卢蒂格（J. Michael Luttig）等法律界人士表示，这是一场“本不该发表的演讲”，它极大地损害了最高法院本应具备的政治中立形象。

	【3】埃博拉新疫情：史上最大规模的暴发正在逼近？<br/>
<br/>
5月21日，NatureNews发文指出，本次埃博拉疫情的规模令人感到担忧，未来几周将决定疫情最终会蔓延到何种程度。<br/>
截至5月15日，刚果民主共和国和乌干达相继宣布暴发埃博拉疫情，官方记录到246例疑似病例和80例疑似死亡。然而短短几天后，一项建模研究提示，实际感染人数可能远高于此。世界卫生组织总干事谭德塞5月19日表示，他对疫情的规模和蔓延速度“深感忧虑”。<br/>
<br/>
与以往疫情相比，当前数字格外引人注目。2014年3月，几内亚最初宣布暴发埃博拉时，报告的疑似病例仅有49例，疑似死亡29例。而这次引发疫情的是一种罕见的邦迪布吉奥病毒，显然已无症状传播数周之久。美国疾病控制与预防中心前主任汤姆・弗里登指出：“这种病毒起步领先”，遏制疫情的努力将面临巨大挑战。<br/>
<br/>
此次疫情发生在刚果民主共和国伊图里省和北基伍省的城镇及半城镇地区，这些地方人员往来频繁、互动密切。5月17日，世卫组织宣布该疫情为国际关注的突发公共卫生事件。谭德塞表示：“鉴于所有这些风险，我决定必须立即采取行动，以防止更多人死亡，并动员一场有效且国际化的应对措施。”但他同时指出，此次疫情并不构成大流行紧急状态。<br/>
<br/>
流行病学家正争分夺秒地查明疫情始于何时。现有证据表明，邦迪布吉奥病毒已传播约两个月。刚果的调查人员已确认一名于4月20日去世的人，暂将其视为首例病例。5月5日发生一起“超级传播事件”后，调查人员发现社交媒体上有关更多死亡病例的报道，他们现在怀疑这些死亡病例是由该病毒引起的。<br/>
<br/>
所有埃博拉病毒均通过接触血液及其他体液传播，可引发类似流感的症状、呕吐和腹泻，最终导致内外出血以及肝肾功能衰竭。邦迪布吉奥病毒株过去曾使30%至50%的感染者死亡，其致死率低于更为常见的扎伊尔病毒株――后者在2014至2016年大规模暴发中致死率约40%，在2018至2020年刚果的一次暴发中更是高达66%。<br/>
<br/>
5月20日，谭德塞表示已有61例确诊病例和近600例疑似病例。然而许多研究人员认为这一数字很可能被低估。根据伦敦帝国理工学院传染病流行病学家露丝・麦凯布及其同事的模型研究，结合以往邦迪布吉奥疫情的病死率以及迄今已登记的死亡人数，实际病例数可能已超过900例，甚至可能超过1000例。麦凯布强调：“至关重要的是，要向受影响地区的民众提供资源，以帮助遏制病毒进一步传播，防止更多死亡病例发生。”未来几周将决定这场疫情最终会发展到多大规模。
</p>

	有人说为什么其他人看电影时都在哭，我看完了没觉得有什么好哭的啊？<br/>
<br/>
这让我想起知乎有个问题「我为什么感受不到演员演技的好坏」，底下有个回答，我在许多年以前就看过了，但还是会时常翻出来看。<br/>
<br/>
尤其是最后一段话。<br/>
<br/>
*******<br/>
<br/>
金庸老先生曾经在《倚天屠龙记》的后记中有这样一段话:<br/>
<br/>
「然而，张三丰见到张翠山自刎时的悲痛，谢逊听到张无忌死讯时的伤心，书中写得太也肤浅了，真实人生中不是这样的。因为那时候我还不明白。」<br/>
<br/>
这段话书写于金庸的大儿子查传侠于美国自缢身亡之后，即便一生命途多舛辗转流离如金庸先生,也只有在至亲之人离世的那一刻才能真正体会到所塑造人物悲痛的肤浅。因此，很多东西我们看不懂理解不了，这太正常了。<br/>
<br/>
很多的天真、无知与狂妄，都来源于人生阅历的缺失，于人情，于世事，于艺术，于生活。<br/>
<br/>
但这并不可耻，也不可悲，毕竟有些阅历如果一辈子都没有，可能会更幸福。

<p>
	看了一篇谢晋的文章，我估计那是个讲演稿，可能谢老说话时，都没带稿子。但思维很清晰，讲地也很有感染力。那是在第三届金鸡奖上的一段谈话。谢导自己的片子《牧马人》当年没拿到什么重要奖项，他当时已拍完《秋瑾》，《高山下的花环》可能是在紧张筹备中。有人提出，要是中越交好，这电影说不定会石沉大海。谢导认为这是瞎操心，我讲的是人，真把战争中的人讲好了，要比讲一个具体的战争，更能留存下来。这话现在听来，道理是通的，但仍会让我们多出一些必要或不必要的感慨来。<br/>
<br/>
谢晋反复提到了《秋瑾》中贵福的扮演者于是之，说他比他舅舅石挥还要好。这话我举手赞成，但又说像于是之这样出类拔萃的艺术家，适合他的电影并不多。也没有专为他的表演，而量身订做一部电影。<br/>
<br/>
说到表演，谢晋特别强调，演员要学会丰富自己，要多在生活中学习。他还特意提到周总理对演员的一些最基本且是最具体的期望：最少你们也要学会开车。在谢晋看来，总理的话，在很多演员那里打了水漂。谢晋说得也很具体，说他电影中的很多演员，都不会骑马，这既影响了表演的质感，也对镜头的调度有所破坏。镜头总想迁就演员并不自如的形体，是拍不好的，拍出来的效果，只能是凑合。他直言，国外的演员要比国内的演员要好很多，他们比我们聪明多少，倒不一定，但他们比我们投入，比我们更舍得自己，却是真的。他们比我们更能享受电影这件事情，而有的演员，到了片场，戏还没拍，人就出了倦态。他们是在吃苦，但吃得最多的苦，是没戏硬找戏的苦。这是没有功劳，还有苦劳，最后也只剩下了疲劳。<br/>
<br/>
而谢晋的另一些说法，是值得商榷的。比如他说电影要表现复杂的生活和复杂的人性，要搁以前，我肯定同意。但电影的形态，应是多种多样的。贪多贪深贪奇，甚至贪真，都是目的走在了前面。一个简单的人，一个不起眼的事，一个人间常见，但很少在人心逗留的仪态，也是值得去拍一拍的。
