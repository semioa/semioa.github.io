	【1】「财新」最新一期封面报道质量挺高的，数据也很扎实，没有简单的把选题苦难化，论述了不少结构性的问题：<br/>
<br/>
- 灵活就业的标准不太统一，但综合庙堂和朝野的多个口径，可以判断灵活就业的人口一直处于增长中，若以人社部的2亿为基准，相当于占到了全国城镇就业人口的47%，已经不能算是就业市场的补充了；<br/>
<br/>
- 显性原因当然是僧多粥少的环境所致，但岗位紧缩并不只和宏观经济有关，民营企业的平均寿命从上世纪九十年代的3.2年下降到目前的2.5年，导致用工周期缩短，以及把「岗位」打散成「任务」的经营意图；<br/>
<br/>
- 把10%的岗位拿出去灵活就业市场消化，可以节省20%-30%的人工成本，显著降低企业的生存压力，实际上餐饮业被学习最多的萨利亚、寿司郎都有把兼职小时工替换全职员工且不降低效率的成功经验；<br/>
<br/>
- 「零工」经济的繁荣不分收入高低，低收入的制造业就不说了，淡季1/3、旺季2/3都是派遣工在干活，高收入的行业里也有医生做「飞刀」、退休专家返聘等趋势渐长，花钱购买一项即插即用的技能、而非雇佣一个完整的人，正在改变就业生态；<br/>
<br/>
- 从就业者来看，30岁是一道分水岭，30岁以下尚有主动选择灵活就业的群体，他们不是找不到全职工作，只是因为对于劳动自由和不被压力的偏好增加，以及对新兴服务市场更加敏感，所以流向了零工市场；<br/>
<br/>
- 30岁以上就没这么从容了，灵活就业是真正意义上的蓄水池，是不得已的生存策略，以网约车司机为例，有77%受访者都是因为丢了工作而选择这一行，其中超过半数的司机是家庭唯一或主要的收入来源；<br/>
<br/>
- 智联招聘的报告显示，收入不稳定（占比54.1%）和兜底性保障较少（占比41.7%）是灵活就业者普遍担心的问题，另有对外卖骑手的摸底调研，愿意接受现行比例缴纳社保的骑手不到10%；<br/>
<br/>
- 灵活就业也分阶层，上层是能自行和市场需求对接、不需要中间商的自由职业者，中层是依附于平台接单的以司机和骑手为代表的新形态劳动者，下层是庞大的日结老哥，也是零工劳务市场的主力；<br/>
<br/>
- 在企业侧，表示「稳定或扩大灵活用工规模」的占比达到51.8%，较2020年的29.3%大幅增长，人力资源平台由此迎来机遇期，撮合短期工的「拼多多」大受欢迎，「企业需要10个零工，3个小时就能对接完成」；<br/>
<br/>
- 主张纠正平台用工「去劳动关系化」的学者表示，平台经济是解决了17%的就业没错，但同时也贡献了32%的劳资纠纷，「中国的劳动法面临的不是要创新的问题，而是能不能回归基本法制」；<br/>
<br/>
- 也有反对声音认为，从历史来看，所谓的标准就业形态从来就不是人类社会的常态，它起源于十九世纪中后期的工业化，但也在不断的被瓦解中，制度需要适应社会，而不是本末倒置；<br/>
<br/>
- 有几亿人从事灵活就业，这肯定是一个理性选择，不必把它耸人听闻化，这一代人都需要承认和接受就业结构的变化，灵活就业提供的也并非都是低质量的劳动需求，不要因此就对年轻人施加羞耻感；<br/>
<br/>
- 青年就业形势严峻和制造业招工困难的并存，是解释灵活就业这个选择合理性的现实依据，「为什么大家宁愿去给人喂猫遛狗，也不愿意进厂打螺丝呢？」这是一个天问。

	【2】为什么吹的神乎其神的波士顿动力机器人销声匿迹了？<br/>
作者：张子昕<br/>
<br/>
人在波士顿红线上，下班了回答一下。这是一个有点长的故事。<br/>
<br/>
谢邀 @ 蝴蝶先生<br/>
<br/>
首先声明，如果你真心认同"波士顿动力用AI生成的/CG"或者"波士顿动力没有产品"或者"波士顿动力是骗钱公司"，我的判断是开除人籍。建议直接退出这个回答然后去从幼儿园重新读起。<br/>
<br/>
至少在2022年之前，波士顿动力在足式机器人的全身运动控制水平上都是一骑绝尘的。他们的技术护城河是一套能够实时解算 whole-body MPC 的控制框架。搞过轨迹优化的朋友们都知道，求解一个包含20+自由度机器人全身动力学的非线性非凸轨迹优化问题有很多种方法，但是要在几十毫秒内得到一个质量不错的解并且能让机器人动起来就是另一个次元的问题了。这里面包含大量的数学技巧和工程细节，是经过很多年的打磨一点一点积累出来的结果。这一护城河在当时看来是非常难以跨越的。<br/>
<br/>
然后2020年，在风景秀丽的瑞士，一个隶属于苏黎世联邦理工（ETH）的实验室整了个大活。这个实验室就是大名鼎鼎的 Robotic Systems Lab (RSL)。RSL其实一直都非常顶级，可能是当年地球上除了波士顿动力以外唯一有能力写出并且部署 whole-body MPC 的组织，并且在 SLAM 方面也有很多重要的贡献。2020年，他们在 Science Robotics 上发表了一篇影响深远的论文：Learning quadrupedal locomotion over challenging terrain。在那个蛮荒年代，世界上除了波士顿动力以及MIT的 Biomimetic Robotics Lab 以外，还没多少人能把四足的控制器做到能够在野外地形上稳健走路。然后，RSL的老哥们直接把他们自己造的四足机器人拉到了瑞士的山山水水各种地形上遛了十几趟。最颠覆的事情是，他们直接把MPC的桌子掀飞了：咱们一点MPC不用，咱们用强化学习。<br/>
<br/>
从那之后，RSL在强化学习这条道路上直接起飞，发 Science Robotics 如同喝水。他们在仿真和 GPU 并行训练方面的工作直接催生了 Isaac 全家桶的诞生，他们开源的 rsl_rl 框架现在已经成为这个地球上几乎所有人形机器人公司训练控制器的基础，包括波士顿动力自己也是打不过就加入了。可以说，没有RSL的贡献，人形机器人行业的爆发还得再推迟一两年。<br/>
<br/>
在这种情况下，波士顿动力就有点尴尬了，特别是在2022年之后。MPC这块香饽饽突然就不香了，RSL的工作不仅牛逼，而且完全开源，让全世界的人们都享受到了强化学习的力量，而后续 RSL+NVIDIA 的持续发力也是直接把足式机器人的 locomotion 问题差不多完全解决了，whole-body control 也不再是梦，都是基于 rsl_rl 做的。然后宇树也崛起了，充分发挥东大的工业优势，把极其便宜的硬件送进了全世界的实验室里，人手一条 A1/Go1，再到后面人手一个G1，研究进程极速加快。波士顿动力的护城河就这么在很短的时间内被填平了。<br/>
<br/>
所谓的技术壁垒就这么被无情碾碎了，选择的重要性不言而喻。老黄说 NVIDIA 要一直有危机感，这是很对的。<br/>
<br/>
所以到底什么样的技术最后能胜出？能够易于迁移和 scale-up 的技术。满足这两点的技术会随意地把别的技术直接创死。举一个随意的例子，你在一个四足上调好了MPC，但你不能保证一样的算法结构在人形上就好使。你在四足上做好了RL，你的 infra 搞好了，改改 reward function 在人形上很快就能 work。现在的 motion tracking RL reward function 更是简单到令人发指。自由度的增多和机器人结构的变化给MPC带来的麻烦远大于RL。<br/>
<br/>
波士顿动力牛逼吗？曾经牛逼过，人家确实是足式机器人的先驱，2019年在CMU的 FRC High Bay 里坐着看着他们让 Atlas 翻跟头的 demo 感叹不如退学去北极开uber的景象还历历在目。而且他们作的 demo 确实也是激励了很多年轻人走入这个行业。<br/>
<br/>
现在呢？还是强的，毕竟工程底子和公司文化摆在那，机器人除了算法之外，对硬件的理解也是非常重要的一个部分，甚至有时候比算法更重要。<br/>
<br/>
销声匿迹了吗？完全错误，你们千万要注意啊，不要"见着风，是得雨"啊。无中生有的东西，你再帮他说一遍，等于你也有责任吧？<br/>
<br/>
处境艰难吗？难。可能除了特斯拉之外，在美国的机器人公司都面临着生产线搭建和生产成本的严峻问题，你不在中国造成本就是下不来（包括特斯拉自己可能也要在中国制造）。韩国现代在这方面来说是个好的靠山。波士顿动力最先产品化的东西是机器狗 Spot，走的是工厂巡检这个赛道。此外还有 Stretch（不是足式），走的是物流这个赛道。今年 CES 刚刚宣布了新的 E-Atlas 量产型号，似乎势头不错，但到底能干什么仍未可知。<br/>
<br/>
未来如何，谁知道呢。<br/>

在那次关于信任与开放的讲话之后仅 10 天，HHS 便撤销了一项已实施 54 年的政策：对新规章制度征求公众意见。这一举措让许多他曾承诺服务的利益相关方失声。肯尼迪草率解聘顾问与专家，通过付费墙媒体发布政策变动，解雇举报人，并主导修订指南与建议――这些修订往往与数十年的既定科学相矛盾，而且经常让他过去曾抨击的行业从中受益。在肯尼迪领导下，美国国立卫生研究院（NIH）关闭了研究空气污染健康影响的项目；HHS 扣压了一份将饮酒与癌症联系起来的报告；美国食品药品监督管理局（FDA）撤回了关于食用某些产品（如生牛奶和二氧化氯）可能造成危害的警告――这些产品被错误地宣传为可治疗自闭症。肯尼迪对 CDC 的改动导致 26 个州拒绝采纳官方疫苗政策指导；而在 12 月，CDC 还在未经征集的情况下拨款 160 万美元，用于在几内亚比绍开展一项疫苗研究，该研究在伦理上引发了如此多的担忧――其设计可能让数千名未接种儿童面临乙肝暴露风险――以至于有人将其与臭名昭著的“塔斯基吉未治疗梅毒研究”相提并论。 ￼<br/>
肯尼迪治下的 HHS 已形成一种习惯：把好钱继续砸在坏科学上。在特朗普政府削减科研经费与人员之际，优先事项出现了有害的转向。从 mRNA 疫苗到糖尿病与痴呆等关键主题，前沿发现与临床研究被剥夺至关重要的资源；与此同时，缺乏正当理由的“垃圾科学”和边缘信念却被抬升、被赋权。在肯尼迪领导下，NIH、FDA 与 CDC 的政治化正在危及美国科学与创新的未来，并扼杀维系当下国家安全的公共卫生体系。 ￼<br/>
联邦政府用于监测与报告健康问题（如药物过量、孕产妇死亡率、粮食安全）的机制，和依赖这些机制的医生与科学家一样备受冲击；数以千计的数据集不再向公众开放，使美国乃至世界在应对未来危机时准备不足。而危机正在逼近：2025 年 11 月，华盛顿州记录到 H5N5 型禽流感的首例人类感染（并死亡）；百日咳在 2025 年导致美国 13 人死亡，目前仍在全国蔓延；而始于去年 1 月的麻疹疫情，如今已威胁到美国与墨西哥的“消除”地位。 ￼<br/>
尽管形势如此，肯尼迪仍持续散布错误信息，并以牺牲国家最脆弱群体利益为代价推动政治化议程。当国会要求他为其决策负责时，他的回应既闪烁其词又充满对抗。肯尼迪在 1 年内造成的破坏可能需要几代人才能修复；只要他仍掌舵，美国的健康与科学就几乎看不到希望。要求他辞职的呼声如今已达数千之众。国会必须履行监督职责，就他的执政记录追究其责任；否则，就等同于接受并承担为特朗普总统让他“在健康领域横冲直撞”的决定背书的责任。<br/>

	【6】美国专家：【情报民主化时代】<br/>
<br/>
仔细看这张图片。你看到的是一张中国商业卫星拍摄的沙特阿拉伯苏丹王子空军基地的照片。每个红框都是人工智能模型按类型识别出的美军飞机。每个标签都是中文。你看到的这个基地就是伊朗周六晚上发射弹道导弹打击的那个基地。<br/>
<br/>
一家名为MizarVision的公司，五年前在杭州成立，发布了这张照片。不是五角大楼。不是中情局。不是递交给战情室的机密情报简报。而是一家中国初创公司，拥有亚米级分辨率地球观测卫星的访问权限，以及能够从轨道上区分KC-135同温层加油机和KC-46飞马加油机的人工智能目标检测模型。<br/>
<br/>
《航空周刊》确认了图片显示的内容。十五架KC-135空中加油机。六架KC-46飞马加油机。六架E-3哨兵空中预警机――这很重要，因为整个美国空军库存中全球仅剩三十一架E-3，意味着美国约五分之一的现役预警机队停在沙特沙漠的单个停机坪上。两架E-11A战场空中通信节点飞机。C-130大力神运输机。C-5银河重型运输机。"史诗狂怒"行动的骨干力量，从太空被编目并发布在微博上。<br/>
<br/>
这就是伊朗打击的基地。法新社驻利雅得记者报道了首都东部发生爆炸，浓烟升起。沙特外交部谴责伊朗针对利雅得和东部省的袭击。沙特防空系统拦截了射弹。但你看到的这张图片是在打击前几天发布的。这意味着伊朗拥有与MizarVision免费向全世界提供的完全相同的情报图像。<br/>
<br/>
这就是情报民主化的样子。1991年，只有美国能够从太空看到停机坪上的单个飞机。2003年，少数国家拥有这种能力。2026年，一家中国初创公司在社交媒体上发布美军兵力部署的注释卫星图像，《航空周刊》在第一枚导弹发射前就进行了分析。<br/>
<br/>
《亚洲防务安全》捕捉到了这意味着什么：亚米级分辨率图像区分单个飞机类型，从根本上改变了打击前部署的保密计算。当商业卫星每天两次拍摄每个停机坪、人工智能模型在分析师喝完咖啡之前就标记每个机身时，你无法在六个 基地集结两百架飞机并保守秘密。<br/>
<br/>
隐藏集结的时代结束了。现在每次部署都可被观察、编目，并由没有安全许可、对任何人都没有忠诚义务的公司近乎实时地以每种语言发布。下一场战争将不会在秘密中策划。它将被每个人从轨道上同时以每种语言观看。

<p>
	一开始没想说觉得是家事，今天聊聊，我公公婆婆挺前卫的。<br/>
<br/>
两年前他们卖掉了住了几十年的房子，分居了......<br/>
不是感情破裂，是想给对方更多自由，更好的享受晚年生活。<br/>
<br/>
卖掉房子我能理解：三层的家老两口确实住不了，打扫卫生都很耗费体力，<br/>
俩人多年前就想处理掉，但家里还有两只猫，已经熟悉了环境，每天要外出，<br/>
他们视宠物为真正的家人，为了它们还是住下来；<br/>
后来两只猫都在20岁高龄寿终正寝了，老两口开始实操分家<img alt="[允悲]" src="/assets/images/191600/006.png" style="width:18px;height:18px;" title="[允悲]"/><br/>
<br/>
我婆婆热爱社交，喜欢便利的生活，喜欢聚会，住到了当地比较高端的一种养老社区，<br/>
就是一个大楼都是老年人，周围环境优美，内部很安静；<br/>
我去过，进去就有一种进图书馆的感觉，<br/>
有整层娱乐设施，比如当地人喜欢的bingo区，国际象棋区、绘画区、瑜伽房等，<br/>
她很热衷参加这些活动，现在足不出户坐电梯就到了；<br/>
她自己的公寓是两居室，有大起居室，温馨的卧室和一个画室，<br/>
不外出她要么邀请朋友来喝茶，要么就在画室画画，自得其乐。<br/>
<br/>
我公公好静，喜欢无拘束的生活，他搬到了乡村小屋，有森林有湖泊，有各种小动物，鸭子天鹅经常来他小院里，有时还能飞来孔雀，<br/>
他喜欢一个人去森林安静的走，然后回家吃不健康美食，喝啤酒。<br/>
哦对了，我婆婆生活超级健康，作息规律饮食严格，78岁了还保持着年轻时的优美线条，<br/>
每天早早起床 ―― 我先生说当时家里规矩就是“起晚了没饭，早餐过时不候”；<br/>
在一起时她还严格控制我公公的饮食，别说油炸和甜食了，饭多吃一口都提醒他“肚子变大了”，<br/>
如今公公自己住特别爽，想吃啥吃啥，想几点起几点起，有时晚上去小镇的酒吧喝点小酒，美滋滋；<br/>
我们去他家里，他打开冰箱得意的展示各种啤酒；<br/>
我先生说他爸确实骄傲，这么多年了没有过，不敢买。<br/>
<br/>
不管谁的生活方式对，俩人似乎都找到了自己喜欢的状态。<br/>
我婆婆不操心了不管了，眼不见心静，<br/>
我公公因为心情好了更自由了，身体也没变差，我觉得他好像还更精神了。<br/>
<br/>
老两口每日通电话，每周约会，要么在彼此家里，带着吃的、鲜花、红酒去看对方；<br/>
要么一起去女儿家、美术馆、会老友；<br/>
过着好像男女朋友的生活，距离产生美，关系还更融洽了。<br/>
<br/>
哈哈其实我挺羡慕。
