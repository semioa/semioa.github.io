	看到纽约客一篇关于天气变化的文章，很长，里面有个令人毛骨悚然的故事。<br/>
一个85岁的马里兰老头，夏天在家门口码头收螃蟹笼，抓到六只蓝蟹，挺开心，准备晚上蒸了吃，结果伸手的时候被金属划出了个平时根本不会多看一眼的小伤口。几小时后他胳膊开始变色，大片的深紫红，像严重烧伤一样，皮肤下面像有东西正在腐烂。后来胳膊开始肿胀，液体在皮肤下面积起来，医生怀疑是食肉菌。这种一旦进入伤口，会破坏血管。血浆往周围组织里渗漏，免疫系统拼命想堵住泄漏，结果局部血流反而被切断，肉开始坏死。他被直升机送到Baltimore创伤中心，医生讨论的不是能不能保住胳膊，而是能不能活下来。<br/>
当时唯一可能有效的方法，是直接把感染的肉切掉。几个小时后他醒过来时，看见自己胳膊里的一部分已经空了，肌肉和骨头直接暴露在外面，基本就是一个大洞。<br/>
这种细菌以前在美国南方以外很少见，现在随着海水越来越暖，已经一路往北扩散。<br/>
文中后来提到，人类之所以没有大规模真菌感染，科学家认为一个重要原因是人体温度太高，大部分真菌受不了98°F。但如果整个地球越来越热，真菌也慢慢开始适应高温呢？<br/>
后来现实里也真的已经出现过超级真菌，耐高温，耐药，在医院里极难清除。文里提到一个耳念珠菌的例子，当时在ICU墙壁、窗帘、天花板，到处都能发现。病房得像处理生化危机一样，全身防护服、紫外线机器、整层消毒。<br/>
文章快结尾还说, 永久冻土融化后，几万年前的细菌和病毒正在重新释放。科学家已经从西伯利亚复活了僵尸病毒（估计已经沉睡了数十万年）。当冰川融化时，冰中的微生物也会流入海洋，而它们会对新的生态系统造成什么影响，目前根本无法预测。<br/>
科学家最担心的，还不是这些古老微生物本身会直接感染人类，而是它们的遗传物质，可能会以人类无法预测的方式改变整个微生物世界。有些地方的冰川融水正在与海水、污水混合，没有人能预测这些微生物DNA最终会流向哪里。 <br/>

	说一件最近遇到的荒唐而气人的事情。我们中文系有一个创意写作专业，毕业作品以小说和散文为主体，授予专业硕士学位，有别于以研究性论文毕业的学术硕士。现在的硕士毕业论文都要送到外面去盲审，结果一位学生被一位评审人打了不及格，按规定要修改后送出去重审。<br/>
不及格的理由是什么呢？这位专家先说小说写得是成功的，但是并不具有论文的基本要素：论点、论据和论证过程，没有文献综述，不符合硕士学位论文的要求。<br/>
拜托！看看清楚这是创意写作的专业硕士，搞搞清楚这个方向的培养方案和毕业要求。我们创意写作专业成立十几年了，还是第一次遇到这种理由打不及格的。我的第一感是这位评审人太马虎了，都没好好审题，拔笔就写。可这就苦了学生，经历过评审不合格的都知道那有多折腾。真是从天而降的冤屈，因为是系统随机派发的盲审，都不知这样瞎评的人是谁，没法找他去说理。<br/>
昨天和朋友们吃饭，大家免不了聊AI，现在啥都是AI做，人类的世纪要结束……我突然灵光一闪：这个评审人会不会是用AI评的，所以才评得如此牛头不对马嘴！否则实在很难解释。<br/>
当然这只是猜想，无从当面对证，但这种事情恐怕会越来越多：AI出题目，AI写文章，AI评审……系统一步步地取代人的位置。这个趋势恐怕难免，但至少要多给活人留些相互看见的机会和自证清白的渠道。

	【3】奥尔特曼说，AI成本已经成了重大问题。<br/>
<br/>
OpenAI 公司最大的 token 消耗者，每月使用1000亿个 token，而且这还不是全球第一。<br/>
<br/>
在周二的企业活动上，首席执行官萨姆・奥尔特曼谈到了公司里 token 消耗量最高的人。<br/>
<br/>
奥尔特曼说，六年半前，OpenAI 消耗 token 最多的人，每月用掉10万个 token。他说，那个人“很可能是当时全球 token 使用量最高的人”。<br/>
<br/>
奥尔特曼说：“今天，6年半之后，这大约就是全球人均平均水平，而 OpenAI 内部 token 使用量最高的人每月大约使用1000亿个 token。”<br/>
<br/>
OpenAI 有一种大量使用 token 的文化。据报道，公司有 token 排行榜，员工有时会在 X 上炫耀自己的高使用量。<br/>
<br/>
OpenClaw 创建者斯坦伯格晒出的截图显示，他在30天内消耗了6030亿个 token。《纽约时报》报道称，OpenAI 员工中有人一周内消耗了2100亿个 token。<br/>
<br/>
OpenAI 放开使用 token 的同时，其他公司正在收紧控制。亚马逊关闭了 token 排行榜；Uber 在首席运营官表示相关支出越来越难以证明合理性后，设置了 token 使用上限。<br/>
<br/>
在小组讨论中，奥尔特曼提到了那个网络梗：“我的公司在第一季度就花掉了我整个2026年的预算，你能把这件事做得更高效一点吗？”<br/>
<br/>
奥尔特曼说，公司会继续改进模型，并探索其他方式，以“用更少支出提供更多价值”。<br/>
<br/>
他还表示，成本问题是相当突然地出现的。奥尔特曼说，2026年初，“这个问题从来没有出现过”。他说：“人们对自己的支出水平完全满意。”<br/>
<br/>
他说，现在，AI 成本已经成了“重大问题”。（商业内幕）

	【5】这是我见过的最荒谬的AI应用了哈哈<br/>
<br/>
一家彼得・蒂尔支持的初创公司，生产牛用的AI项圈，现在估值20亿美元<br/>
<br/>
而且我越读越觉得酷。它的原理是这样的：<br/>
<br/>
每头牛都戴着一个太阳能供电的项圈，它与无线电塔网络和农场主手机上的应用通信<br/>
<br/>
农场主不用建物理围栏，而是在应用里的地图上画出围栏，项圈用GPS让每头牛待在那个隐形边界内<br/>
<br/>
当一头牛靠近边缘时，项圈会播放声音来引导她，一个轻微的振动告诉她该往哪走。<br/>
<br/>
这就像汽车在你倒车靠近墙时会发出哔哔声<br/>
<br/>
牛几天下就能学会这些信号<br/>
<br/>
所以现在牧场主可以通过在地图上滑动围栏，把整个牛群移到新鲜草地上，而不用开车出去开一个栅栏门<br/>
<br/>
而且同一个项圈一直在读取每头牛的身体数据。<br/>
<br/>
它每秒对每头动物取五次读数，所以AI能在人步行巡视田地时还没注意到之前，就捕捉到生病、受伤、准备交配或即将分娩的牛<br/>
<br/>
所以这基本上就像给牛用的WHOOP设备了哈哈<br/>
<br/>
他们给背后的AI起了个完美的名字：Cowgorithm<br/>
<br/>
它训练于超过70亿小时的真实牛行为数据，这就是为什么Halter称数据是它的真正资产和护城河。<br/>
<br/>
他们比任何人都更了解正常牛的样子，所以能立即标记出异常的那头<br/>
<br/>
它已经在新西兰、澳大利亚和美国多个州的超过100万头牛上使用。<br/>
<br/>
加利福尼亚甚至在公共土地上用它来放牧牛群，以清除干灌木并减缓野火蔓延的模式<br/>
<br/>
每头牛每月成本大约5到8美元<br/>
<br/>
一份过去意味着铁丝网、栅栏门和整天开车巡视田地的工作，现在基本就是一个人的手机操作

	【7】没有人告诉她要笑。也没有人去求那头骆驼。<br/>
<br/>
📸 2003年，摄影师韩承利正在内蒙古――中国北方这个有超过四百万蒙古族同胞生活的自治区――旅行。在草原上，他捕捉到了这一幕。小女孩名叫布特德玛，那年五岁。这头骆驼是她家庭日常生活的一部分，就像双峰驼几千年来一直是这一地区游牧家庭的伙伴一样。<br/>
<br/>
🖼️ 韩承利把这张照片命名为《蒙古娃》。他把它送去参加比赛和展览。2017年，它成为波兰一个展览的一部分。在Reddit上，许多看到这张照片的用户以为它拍摄于二十世纪初。当有人澄清这是2003年的照片时，最常见的回复是：“2003年？我还以为它有八十年历史了。”<br/>
<br/>
⏳ 这说明了这张照片传达出的某种感觉。它不像是一张二十一世纪初的照片。它像是一幅超越时间的画面――在那个世界里，一个小女孩和一头骆驼可以在草原上一起开怀大笑，谁也不需要解释为什么。<br/>
<br/>
🐫 双峰驼在四千多年前就被中亚地区的人们驯化了。在蒙古，家中的骆驼既是役畜，也是伙伴，它们从小就和人一起长大。孩子们从出生起就认识它们。照片里的小女孩不是在摆拍。她只是在做一件平常都会做的事。<br/>
<br/>
📅 2014年，也就是原片拍摄十一年后，有人找到了布特德玛，在家乡同样的景色中为她和她家人拍下了照片。那个五岁的小女孩已经长成了十六岁的少女。环境没有太大变化。草原依然是草原。<br/>
<br/>
🌍 2003年的那张照片已经出现在全球数百万块屏幕上。好几个人说过，他们把它打印出来，钉在办公桌前或挂在家里墙上。有个人把它在办公室里放了好几年，每天早上都会冲着它打招呼：“早上好，布特德玛。”<br/>
<br/>
🎯 韩承利捕捉到了一些无法计划的东西：两张张开的嘴，在同一瞬间，以同样的角度，在茫茫天地之间。<br/>
<br/>
