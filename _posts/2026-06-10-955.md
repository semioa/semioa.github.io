<p>
	美东时间5月16日上午，国内已经到了晚上了。想起60年前的五一六，是一个值得记住的日子。那么，再往前推60年。1906年的中国在做什么？码在一起比较一下，很有意思。<br/>
1966年春节在1月21日，是有史以来罕见的春节最早的一年。春节早说明春天也会早一点，但是，这个春天有点冷。<br/>
3月8日5时29分14秒，河北省隆尧县北纬37度21分，东经114度55分发生震级为6.8级的大地震，震中烈度9度强。<br/>
3月22日16时19分46秒，河北省宁晋县（北纬37度32分，东经115度03分）发生震级为7.2级的大地震，震中烈度10度。<br/>
3月27日，刘少奇和陈毅出国访问前在新疆作短暂停留，先后听取了自治区党委的工作汇报，视察了和田、喀什和克孜勒苏等地。他们不知道，一场风暴在等着他们。<br/>
5月4日～26日，中央政治局扩大会议召开，通过《中国共产党中央委员会通知》（简称“五一六通知”） ，文化大革命正式启动。<br/>
5月29日，清华大学附属中学成立了第一个红卫兵组织。<br/>
5月31日，陈伯达率领工作组到人民日报社夺权。<br/>
这个春天是以夺权为标志结束的。<br/>
夏天又热的不得了。<br/>
6月1日，《人民日报》社论，号召“横扫一切牛鬼蛇神”。同日，北京大学聂元梓等人的一张大字报，向全国广播。<br/>
8月1日～12日，中共八届十一中全会召开，通过《中国共产党中央委员会关于无产阶级文化大革命的决定》。<br/>
8月5日，毛泽东的“炮打司令部”大字报发表。<br/>
8月16日，全国性的大串联开始。<br/>
8月18日，毛泽东第一次接见全国各地的红卫兵，并戴上了红卫兵袖标。<br/>
由最高领袖亲自发动和主导的全国性大乱，举世罕见。<br/>
那么，再往前推60年，1906年的中国在做什么？<br/>
也在改革和动荡中。<br/>
2月，第二次南昌教案爆发，法国主教被杀，群众捣毁教堂，外交交涉激烈。<br/>
 2月，秋瑾在上海创办《中国女报》，倡导女权与革命。慈禧太后面谕学部，实兴女学。推广新式学堂，女学正式开禁。<br/>
3月，清廷禁止人口买卖、停选八旗秀女，社会改良起步。<br/>
9月1日，清廷颁布《宣示预备立宪谕》，宣布预备仿行宪政，派五大臣出国考察归来后推行。<br/>
10月27日――清廷命各省兴办图书馆、博物院、动物园、公园。<br/>
12月4日，同盟会首次大规模起义<br/>
萍浏醴起义（江西萍乡、湖南浏阳、醴陵）爆发，同盟会刘道一、蔡绍南策动，矿工+会党约3万人，号称“中华国民军南军”。<br/>
12月底：起义失败，刘道一、魏宗铨等被杀，万人殉难；为同盟会成立后首次武装起义。<br/>
两个甲子，总是在改革，但是，为什么总是进步不快呢？

	李翊云的访谈其实已经解答了很多了，但她的书其实覆盖了更多局外人想象不到的维度（当然也包括局外人能轻易想象的维度），包括为什么还要写，为什么要此时写，在两个儿子的养育过程中遇到不同的问题。鉴于两个小孩的资质，曾经甚至有很多人认为她的教育方式太保守了。在我看来，她已经在普通人所认为的“安全”、“爱”和“尊重”的角度做出了极大的平衡。<br/>
现在的批评之声，无非就是想证明她的教育很失败；她没有遵守普遍的mourning的流程；她再怎么写也是她的一面之词（大家可学会“巧言令色”这个成语了）。如果把文字当成辩护、当成排解、甚至当成简单的记录，它当然是不完美的，是充满矫饰的，但李翊云在书里提出了placeholder这个词，“a sentence is a placeholder for a real sentence, or that a thought is still at a place-holding stage, waiting for a clearer thought to emerge in sharper words.”文字是求真的第一步，是进入那个谜团、迈入那个命运的第一步，是接受自己将终生待在一个没有出路的abyss里的第一步。一个职业作家，对文字当然首先是信任，甚至是信仰。言必称“巧言令色”，无非是掩饰自己不想写、不会写、写不好的软弱罢了。

	【7】刚刚『连线』杂志发表了一篇有关 Meta 内部现况的特稿，读完很是感慨。<br/>
<br/>
曾经被无数年轻人向往的顶尖大厂，如今却成为了大家最想逃离的炼狱。<br/>
<br/>
5 月 20 日，Meta 将正式裁撤 8000 名员工，但是裁员名单却始终没有公布。加上最近 Meta 为强推 AI 而搞出的很多魔幻操作，以至于 Meta 内部的情绪已经跌到了历史冰点。<br/>
<br/>
这个特殊的时刻，『连线』杂志采访了16名 Meta 员工，从他们的视角描述了最近发生的一切：<br/>
<br/>
- 过去四年，Meta 已累计裁员约 2.5 万人。最新这次裁员其实从 3 月就开始在内部流传，但公司连续数周没有任何官方回应。直到上个月的全员大会上，扎克伯格才正式开口。他的大意是：AI 研发极其烧钱，他别无选择。<br/>
<br/>
- 在等待裁员名单的这几个月，所有员工都活在惶恐之中。害怕被裁员的人只能拼死拼活地赶项目，试图证明自己凭什么值得被留下。<br/>
<br/>
- 有退路的员工则普遍希望自己出现在裁员名单上，因为被裁意味着至少 16 周遣散费加 18 个月带薪医保，主动离职则什么都没有。一位 Instagram 员工说：“大家的唯一念头就是，赶紧给我个痛快吧。”<br/>
<br/>
- Meta 今年连续第二年缩减年度加薪中的股票激励，在去年削减 10% 的基础上，今年又砍了 5%。员工总薪酬中位数已从 2022 年的 41.74 万美元降至去年的 38.82 万美元，加上公司股价今年下跌约 5%，对薪酬里有一半是股票的员工来说，实际收入的缩水相当明显。<br/>
<br/>
- 与此同时，Meta 一季度利润近 270 亿美元，总支出同比增长 35% 至 334 亿美元，扎克伯格去年为几位顶尖 AI 研究员开出了高达 1 亿美元的年薪。用扎克伯格的话说，Meta 不是没钱，而是需要让人力为更远的 AI 雄心提前让路。<br/>
<br/>
- 大约同一时间，Meta 在美国员工的工作电脑上强制安装了一款追踪软件，记录每一次键盘敲击和鼠标点击，目的是收集数据训练 AI，让模型学会像人一样操作电脑。<br/>
<br/>
- 16 位来自不同岗位的员工向《连线》证实，这项计划没有任何退出的选项。当员工在内部群组抗议、并翻出 Meta 泄露用户隐私的旧账时，CTO 博斯沃思并未选择安抚，而是对异议者进行了贬低和训斥。<br/>
<br/>
- 据知情人士透露，扎克伯格在内部解释说，公司之所以监控自家员工而不是外包人员，是因为自家员工『更聪明』。<br/>
<br/>
- 本周，一个小规模抗议团体在美国多个办公室张贴传单，号召同事联署请愿，要求停用该监控软件，并在请愿书中表示这『完全背离了 Meta 标榜的负责任 AI 原则』。<br/>
<br/>
- 一位政策部门员工表示，对公司使命的认同感已经消失，美国本土员工感觉自己是在为最终会取代自己的 AI 模型充当数据耗材。<br/>
<br/>
- 上个月初，至少 1000 名顶尖工程师被强制转入新成立的 Applied AI Engineering 部门，负责为 AI 研究团队开发工具和数据，拒绝转岗的人将面临被裁的风险。在硅谷，这种做法相当罕见 ―― 通常架构调整时，工程师可以自主选择去向其他团队。员工私下将此事称为『抓壮丁』。<br/>
<br/>
- 今年 3 月，加州与新墨西哥州法院分别作出独立判决，认定 Meta 的产品缺陷与政策失误导致了用户的负面体验，责令其支付总计近 3.8 亿美元的赔偿金与民事罚款。多位员工表示，这两起判决印证了他们内部团队长期以来的道德担忧。<br/>
<br/>
- 在英国，部分 Meta 员工正在征集签名，筹备加入英国最大的科技工会，以寻求对工作、福利和隐私的保障。内部组织者在致同事的公开信中写道：“管理层正在变本加厉地采取残酷且短视的行动，我们必须制造筹码，倒逼他们把我们当人看。”<br/>
<br/>
- 公司内部的处境因岗位不同而相差悬殊。负责前沿 AI 研究的 TBD Lab 几乎未受这波动荡波及。而对其他员工来说，AI 更多以压力的形式出现 ―― 绩效考核开始与『在部门内推动自动化的力度』挂钩，员工被要求积极拥抱 AI，不少人坦言自己是在为了用 AI 而用 AI，硬凑出自动化发邮件或自动化写报告之类的成果。<br/>
<br/>
- 扎克伯格在财报电话会议上提到，以前需要几十人花几个月完成的项目，现在一两个人不到一周就能搞定，他表示正围绕这些人打造『公司的下一次进化』。一位资深高管也坦言：『未来公司里一定会有一整个部门的自动化效率碾压人类，这很残酷，这些人注定会失去工作。』

	【8】斯坦福大学神经科学家罗伯特・萨波尔斯基证明，慢性压力是医生们常常忽视的隐形杀手
</p>
<p>
	在克里斯・威廉姆森的播客节目中，他揭露了10个你每天都在做的“正常”习惯，这些习惯会破坏你的睡眠、情绪和神经系统。<br/>
<br/>
1）在脑海中重现对话。萨波尔斯基的原话是：“反复思考15年前你对老师说的那句奇怪的话。”你的大脑并不知道那次对话现在并没有发生。它会触发与最初威胁相同的皮质醇激增。<br/>
<br/>
2）担忧尚未发生的情景。萨波尔斯基的经典例子是：斑马对狮子的反应只有30秒，然后就平静地吃草。而人类却会为想象中的狮子担忧30年。当你意识到自己有这种倾向时，问问自己：“这件事现在正在发生吗？”<br/>
<br/>
3）浏览负面新闻。萨波尔斯基：“当你读到乌克兰或孟加拉国正在发生的事情时，你会激活1.5亿年前进化而来的、用于躲避捕食者的应激回路。”<br/>
<br/>
4）沉迷于悲剧或暴力内容 萨波尔斯基：人类在观看屏幕或阅读小说时，会独特地激活完整的应激反应。你的神经系统无法区分虚构与现实――无论哪种情况，都会导致皮质醇飙升。注意你晚上7点以后的摄入内容。<br/>
<br/>
5）试图控制你无法控制的事情 萨波尔斯基证明：缺乏控制以及试图控制无法控制的事情都会导致慢性皮质醇飙升。无论哪种情况，你的神经系统都会一直处于生存模式。每天放弃一件事。练习这种差异。<br/>
<br/>
6）因为“太忙”而忽略社交联系 萨波尔斯基的研究表明：缺乏社会支持是导致慢性压力的五大因素之一。拥有牢固社交关系的狒狒皮质醇水平更低、寿命更长、免疫力更强。像安排会议一样安排社交活动。<br/>
<br/>
7) 把减压留到周末。萨波尔斯基的原话是：“这不是周末才做的事，而是你每天都要抽出时间来做的事。”每天20分钟胜过周六3小时。像安排会议一样，把减压时间安排好。<br/>
<br/>
8) 强迫自己冥想，即使你讨厌它。萨波尔斯基说：“如果我花20分钟冥想，到周末我可能就中风了。确保你做的是你真正喜欢的事情。”如果你的减压方法让你感到压力，那它就是错误的方法。试试散步、听音乐、培养爱好。<br/>
<br/>
9) 不断追求下一次多巴胺刺激。萨波尔斯基说：“我们是贪得无厌的物种。”多巴胺与奖励无关，它与期待有关。昨天感觉很棒的事情，明天可能就感觉不够了。注意你何时在追逐。停下来。<br/>
<br/>
10）待在让你精疲力竭的环境中。萨波尔斯基对狒狒的研究表明：在不稳定的狒狒群体中，低等级的狒狒皮质醇水平高、端粒短、寿命短――即使它们吃饱穿暖、安全无虞。有毒的工作环境也会对人类造成同样的影响。仔细审视一下谁最消耗你的精力。<br/>
<br/>
慢性压力会抑制大脑中负责同理心的部分――前扣带回皮层。你会变得不那么宽容、不那么慷慨、更加自私。压力大的人并非坏人，他们只是精疲力竭。
</p>

	【9】心理学上有个词叫身份重置效应。当你去到一个没人认识你的地方，就可以重新选择自己想成为的样子。那些在家乡被固定的身份，被旁人期待的人设，到了异地都会全部清零。因而旅行让人上瘾的从来不是风景，而是这种“可以不是自己”的自由感。旅行的意义也不只是看世界，更是暂时不当那个被世界看惯了的你。
