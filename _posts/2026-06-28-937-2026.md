	关于我朋友创业失败负债几百万的故事3<br/>
2022年2月，黑黑拿下了他想要的场地。<br/>
700平，上海世博源，一个巨大的空间――能做出他脑子里那个“桃花源”的最低配置。更牛逼的是，他还谈到了4个月免租期。<br/>
聊场地不是黑黑擅长的事。在 pulupulu 的年代，他是纯粹的产品设计师：只需要把城堡造出来，把机关做到极致，让玩家玩到尖叫就行。至于融资、场地、物业、政策、合同――他从来不需要操心。<br/>
但这次不一样，这是他的创业项目。 <br/>
他必须把所有“不擅长”的事也做起来，像一个突然被拉去打综合格斗的人：你本来只会漂亮的一拳，现在你得学会摔跤、学会护头、学会挨打还得站起来。<br/>
他成长的速度令人惊讶。他说：创业嘛，时间就是生死线，每一分钱都要花在刀刃上。装修期预估4个月，所以必须2月把场地签下来；再留1个月试营业，暑假开场，把第一波口碑做爆。<br/>
听起来多么合理，甚至有点像命运都在配合他。场地方本来根本不想租给他，但是最终那个国企领导被他的真诚打动，市中心、交通便利、租金合理、超长免租期……写进商业计划书里都闪着金光。<br/>
然后，现实用最粗暴的方式告诉你：计划书只负责好看，不负责兑现。<br/>
免租期本来是创业者的护盾，但封控是穿甲弹，直接把护盾打穿。桃源这局真正的第一刀，不是产品，不是营销，是两个字：空烧。<br/>
2月刚签完场地，黑黑和装修队去现场转悠了一圈，盘了一下布局思路，回家就封住了。再能进场，已经是8月中下旬。<br/>
想象一下那种荒诞感：你明明拿到了700平的“世界入口”，却只能隔着空气看它。你所有的热血、设计、演员、世界观，都卡在一扇打不开的门外。 <br/>
而时间在里面滴答滴答地吃你的钱。<br/>
免租期当然被延了一点：场地方后来又额外减免了一个半月，听上去还挺“讲理”。但这跟他们最初的节奏完全不是一回事。免租期本来是用来争取施工的，结果全耗在等待里。<br/>
现在免租期没了，装修才刚刚开始。你要推进项目，就得装修；你要装修，就得承担接下来几个月的硬成本。<br/>
租金是17万一个月。 <br/>
凭空多出来的大几十万，像是有人把你的底池突然加到你无法弃牌的程度。你能扛住吗？还是现在就认栽――订金不要了，赶紧止损？<br/>
当时其实还有一个选择：去和场地方谈判，把钱要回来，至少能要回一半。但他们没这么做。<br/>
他们去考察市场，看见各大商圈呈现出一种诡异的繁荣：大家被憋坏了，报复性消费的苗头很明显。那种热闹很容易给人一种错觉：疫情可能要结束了，一切会回到正轨。<br/>
于是黑黑开了个讨论会，把合伙人、投资人都拉进来。大家说的话几乎一致： <br/>
“相信你，黑黑。”<br/>
“你能把产品做好。”<br/>
“你以前做出过《时间裂痕》，这次也一定行。”<br/>
我说过，黑黑不是那种一意孤行的人，他甚至很依赖这种“集体讨论”来确认自己没有走偏。 <br/>
可创业里有个残酷事实：讨论并不会分摊责任，讨论只会分摊情绪。 <br/>
最后真正要签字、要扛结果的，永远只有自己。<br/>
更现实的问题是：原来的预算里根本没有这笔“空烧”的成本。<br/>
 事已至此，没有别的办法――只能自己投钱。这就引出黑黑人生里最重要的人：雪猫。<br/>
她是黑黑的妻子，当时在一家咨询公司做高管，收入不菲，是那种很能撑事的女人。黑黑把项目实情告诉她，雪猫几乎没有犹豫：“我支持你。”<br/>
她先拿出了积蓄，但很快发现不够。<br/>
公司名义贷不出款，只能个人上。黑黑能贷的额度有限，而银行更喜欢的“抵押物”，是他们的房子，以及雪猫稳定的现金流。于是雪猫也签了字――连带担保，把项目的风险第一次从公司搬进了家庭的资产负债表。<br/>
黑黑在银行签字的时候快要哭出来了，一直问：“你们确定吗？”<br/>
所有人都说：我们相信你。 <br/>
雪猫也说：我相信你。<br/>
几年后我问雪猫：你怎么会同意把创业风险转嫁到家庭里？你怎么能让连带担保击穿你们家的资产负债表？ <br/>
雪猫沉默了一会儿说：“我那时候对家庭资产表的认知几乎为零。他需要钱，我有钱就给；他需要签字，我就签了。”<br/>
她坦诚地说：我们所有人表面上都说是基于对黑黑的信任，但我们始终把他当成唯一负责人。大家一起讨论，其实并没有真的把风险当成“自己的选择”。她当时以为自己是在提供无限的情感支持，回头看才发现――那也是一种无限的责任推卸。<br/>
到这里，桃源已经变成了另一种东西。 <br/>
它不再只是一个“开放世界”的理想，开始变成一张越来越复杂的借条，一份越来越沉的责任。

	唢呐高手大舅却因空耳被嘲笑了40年<br/>
在上个世纪我的老家，发展出来一整个独特的挣外快方式，就是在通往火-葬-场的路上每间隔几十米站着一个中年大叔吹唢呐，只要送-葬的队伍一露头外快小部队就会一个接一个的开始整活，送-葬的家属也会给大人一盒烟或者1-2块钱，如果碰到大户家庭就会给5块钱和一盒好烟。<br/>
由于大舅音乐天赋极高（如果好好培养多好）很快就学会了一大段悲伤送走曲唢呐，我大舅因为眼神好一般站在比较前面的位置，由于头里人少而且大舅年轻气足吹的响，经常会得到更多的赏钱。<br/>
常在河边走哪有不湿鞋，年纪轻轻的大舅也没想到自己命里会有次一劫，前几天的大暴雪压坏了火葬场上面村落的电线杆导致2个村子没电，为了尽快帮助村民恢复通电便找了一辆马拉大板车（因为雾太大开车能见度低雪天路滑不好走所以找了大板车），拉上维修道具和十几名维修工人从火-葬-场下面上去帮助村民维修。<br/>
但这个大板车和当时殡葬用的大板车为同款大板车，因为东西比较重所以还是三匹马在拉的加大大板车，清晨大雾天远远的看过去就像是首富搞的超级加倍大殡-葬，这对于外快小分队来说那可是万年难得一见大活，所以便从第一个看见的人开始就用了吃奶的劲儿开始吹。车越近吹的越响，看清了是三匹马就吹的更响，看清了车上都是人恨不得多长出来一张嘴来吹，等看清了车上没有棺材只有变压器后震天的唢呐戛然而止。<br/>
维修小队在一路悲伤送走曲的伴奏下，加上早起的起床气加班的怨气，愤怒值不断地上涨，维修小队就化身成了一个一点就炸的炮 仗，爆 发就差一把火，幸运的是那个火种就是我大舅。<br/>
那天雾很大人很少大舅也想狠狠赚一票，等我大舅看见大板车冒头开始便玩命的吹，那声音盖过4-5个唢呐根本不是问题，过大的声音也增加了刺激，电力工人们便有2-3个跳下车来，想着跑过来告诉我大舅别吹了，我大舅看见人来了，以为跑过来给他送赏钱了，这下更卖力了，疯了一样的吹，看见人家边跑边比划手势大舅灵机一动，想着哭丧是不是能给的更多，便扔下了唢呐扑通跪在地上开始死命的嚎还用手拍地以表哀伤（我大舅为啥没学表演，这不是影帝，谁是？）<br/>
这时，工人也跑过来了，愤怒的把我大舅一把薅起来，愤怒的说“别嚎了，看清了我们是电力的，不是送-葬的”。但当时大舅带着棉帽子情绪也过于投入，便过滤掉了重要信息只记住了是电 力的，便继续嚎“官老爷走好啊~啊……”<br/>
这一句可是点了工人们的炮了，不仅哭丧要钱还给头儿送走了，便给我大舅一拳，大舅被突如其来的小拳拳打蒙了怒吼道“咋了还打人，是不是不想给钱”<br/>
工人也急了骂道“还给你钱，我给你个baba”然后便扭打到了一起，虽然我大舅年轻力壮但是势单力薄，不仅被一套组合拳安排了还把他的外快道具小唢呐给撅折了。（其实没有下重手只是撂倒了我大舅然后撅折了小唢呐）<br/>
但对于大舅来说这是受到了极大的侮辱，便拿着折了的小唢呐回来找了大舅姥爷，大舅姥爷一听儿子被人给打了气的火冒三丈，被打的理由还是因为不给赏钱，气的带上儿子找到了电-业-局的书记让他给评评理。<br/>
书记不愧是领导，在我大舅的愤怒投诉后冷静的打了一个电话了解了真实情况，之后还和他们讲明了事情，还说了他们打人是不对的但他们沿路吹丧乐讨赏也是不对，并登记了信息说打人的员工会登门道歉，还会赔偿我大舅一个新唢呐。<br/>
经过领导的安抚我大舅姥爷也很平和满意的带着我大舅回家了。你以为这就结束了吗，哈哈哈哈哈并没有社死的现场这才真的来了，因为事后工人登门道歉发现家里面没人，回来便写了致歉信，就带着致歉信和唢呐送到了大舅学校的警卫室交给了校工，校工以为是大舅获得了电业局的表扬，因为唢呐上还绑了一个红丝带，就把东西给了校长，校长就当做光荣事件拿到了第二天的早操上面宣讲，并让大舅朗读了信件，信件写的很委婉：“因为误会，不小心损坏了你的唢呐，对不起，至此特附上致歉信和新唢呐，向你表示歉意”<br/>
老师和校长还问我大舅你原谅工人叔叔的无心之失了吗，大舅还很感慨的说自己是成熟的学生并不会把这些小事儿放在心上。<br/>
校长还夸奖了大舅，但接下来的一句话让大舅终身难忘。既然litao（大舅的名字，为了怕被认出来就打拼音代替）同学原谅了工人叔叔，误会也就解除了，那就让litao同学吹奏一曲送给工人叔叔，告诉工人叔叔您辛苦了。<br/>
那一刻大舅的血液仿佛凝固了，因为他除了那首悲伤送走曲并不会其他，而且他现在骑虎难下，于是就推脱说“我还不怎么会吹”，那一刻校长仿佛上头了非要让大舅吹一首那感觉就是，litao你今天吹也得吹不吹也得吹，不然就是不给我面子。<br/>
大舅于是在校长和老师期待中还带着点威-胁的眼神中，当着全校同学的面吹响了拿手悲伤送走曲，起初校长和老师们还享受其中，但不一会儿就发现了其中的不对这个曲儿明显是不对头，便小声在大舅耳边说别吹了快停下，但大舅的空耳技能再一次发挥出来以为说的是大点声别停下，于是心一横更使劲儿的吹了起来。<br/>
于是校长的愤怒值也被点燃，大声说道可以了谢谢litao同学，然后大舅的班主任就很有眼色的带大舅退场了回教室的路上还边走边说以后你可学点好吧，自这之后的每年家庭聚会和同学聚会大舅的故事都为我们带来了很多的欢乐。

	京津冀地区的蓝天非常可贵，这是整个地区全社会共同努力的结果。但好事应当办好，现在河北一些地区的农民因用不起清洁能源而忍受寒冷，这决不应该，这个问题需要全社会义不容辞地共同帮助解决。过去烧煤烧柴的农民改用天然气和电增加了成本，这个成本理应由整个地区从蓝天中受益的所有机构和居民共同承担，我相信，这也是国家政策的指向。<br/>
如果河北的农民们因为不能继续烧煤烧柴了，烧气用电又花不起钱，只能在数九寒天温度过低的房间里瑟瑟发抖，这决非整个地区煤改气煤改电政策的初衷。那样的情况下，京津冀地区享受蓝天的大城市居民们都会愧疚的，因为我们欠那些河北农村父老乡亲的。<br/>
对河北农民烧气的取暖补贴决不应后退。农民烧气的总费用不应该高于之前烧煤烧柴，最开始推行烧气时的补贴是多少，今天应当继续补贴大体相当的比例。看到主流媒体的报道说，河北有的地区把农民烧气的补贴减了80%，导致那些农民用气的价格比北京和天津农村的煤改气专项采暖用气价格都要贵，这显然不合理。<br/>
据主流报道，河北农民100平米的住房，一个采暖季需要花费7560-11340元，而河北农民是整个地区收入最低的群体之一，2024年河北农村人均可支配收入只有22022元，上述采暖费用相对于他们的人均收入来说实在太高了。这个费用即使在北京、天津的城市居民中，也会有很多人家感到难以承受。<br/>
农村住房都是单门单户的，采暖的能源消耗显然要比楼房里同平米的公寓大得多。这种情况下，就应该由整个社会给农民使用清洁能源取暖提供足额补贴。我们的社会不能得到了蓝天，却对补贴农民补贴抠抠搜搜的，把煤改气的巨大成本甩给农民自行“消化”。<br/>

我认为，补贴农民取暖不应看成负担，而应成为推动消费升温的积极举措。我们的治理应当有能力有办法打通这当中的堵点，实现促进经济繁荣和民众幸福感同时上升的多重效果。<br/>
北京市和天津市都被河北省环抱，头顶同一片天空，是河北广大农村煤改气的受益者。老胡认为，这两座城市应当为减轻河北农民因煤改气改电取暖成本上升做出贡献。

<div class="detail_wbtext_4CRf9">
	刚才转了一个关于现在的电影为什么越来越假的视频，其实不只在电影里，小说世界存在同样的问题，我的判断是，一种叫“光滑”的无形魔怪，入侵了人类世界。它要让一切光滑起来，让一切无瑕，让所有语句、节奏、气息都带上高度克制的光滑。这是为人类进入虚拟世界前做的准备。<br/>
所谓光滑，一种经过抛光的现实，比现实高两三厘米，没到高不可攀的地步，但又绝非无心插柳，是经过精心计算的提升。提升的结果，是让被描述的对象仅仅停在浅表，不让观看者介入，因为它经不起介入，后面什么都没有。<br/>
光滑就是这么一种东西，并非出于自然，而是出于人工，显示着工艺水平的高低，城市化的进展。一件光滑的东西拿在手里，让人联想起的是打磨的过程、工序的繁多和工时的漫长，要去除毛刺，去掉棱角，去掉一切无关紧要的细节，它体积不大，显得很轻，但密度很高，拿在手里有一种意外的分量。它得有一种恰到好处的冷，恰到好处的拒人于千里之外，但它的冷只是一点凉意，还没到冰冷的程度。要达倒这种效果，需要许多条流水线的合作，凝结几代人的劳作，但它一点都不声张，就是那么光滑着。<br/>
邓丽君是光滑的，伊莲・佩姬的声音是光滑的，iPod和iPhone的后盖是光滑的，欧洲的玻璃制品是光滑的，简・奥斯汀是光滑的，伊恩・麦克尤恩的小说是光滑的，托宾的那种厌倦是光滑的，梦枕貘的古代是光滑的，珍妮特・温特森即便在最阴郁的时候也是光滑的，拉斯・冯・提尔的电影是光滑的，埃里克・侯麦的絮絮叨叨也是光滑的。<br/>
光滑是城市文明的成果，三十年以上没有战争、没有饥饿的日子，才能促成这种光滑，这种光滑像一种魔术，突然间就出现在某个时代，像特效一样在某些器物、某些脸庞、某些人物、某些话语中浮现，这种光滑也特别易碎，特别脆弱，只要几声枪响，几声粗野的吆喝，它就又像特效一样褪色了。<br/>
刘慈欣的《三体2黑暗森林》里，来自三体文明的探测器，在2212年，和人类的战舰短兵相接，随后摧毁了人类的整个舰队，因为探测器的外形像一粒水滴，因此被人类称为水滴。后来，王壬用一个名为《水滴》的14分钟短片，还原了这段场景，画面从微观物质开始，慢慢拉远，一直拉到整个舰队，而这一切，都映照在水滴外表“绝对光滑的全反射镜面”上。人类舰队的粗糙和水滴无懈可击的光滑形成鲜明对比，那种光滑，被用来衡量两种文明的差距。<br/>
从工艺水平上讲，那种光滑不难达到，难的是我们在那种光滑里觉得理所当然。四方田犬彦先生曾经写过一篇与《冬季恋歌》（2002年）有关的文章，在谈到这部由裴勇俊主演的电视剧为何会受到韩国观众热爱时，他特意列出一张时间表，将故事里的时间，和韩国的现实时间进行比较，结果发现，这部剧的时间跨度并不小，但同时期韩国发生的大事都没有出现，剧中人完全没有被社会的剧烈动荡影响到，只管在一个真空般纯净的环境里恋爱和思念。<br/>
这种影像中的纯净环境，不难炮制，难的是我们无法在这样纯净的环境里沉浸很久，因为更广大的世界还异常粗粝，这让人很难心安理得。<br/>
也许，冥冥之中，另有一只手，正在没日没夜地打磨着我们周遭的一切，当然，打磨的时间要久一些，至少会久过韩国。但是不管怎样，先光滑起来，其余的，在进入光滑世界之后，都可以补齐。<br/>
我们总算都在奔向那个光滑世界的路上，没有拖延，没被中断，我们无法计算抵达那里需要多少时间，但对光滑之美的渴望，已经占据了一切。
