	前几天，B站的up主 『小A在上网』的视频被全网下架。<br/>
<br/>
小A是我长期关注的所谓“躺平”UP主里，最让我欣赏的一位。<br/>
<br/>
第一 他不卖惨，视频内容里从来不来那套“好赌的爸，病重的妈”，并不想用自己生活多惨来博取同情然后等着那个路过大哥喝高了一心软就爆金币<br/>
<br/>
第二 他不刻意造人设 有好几个UP主，明明整了几十万甚至上百万了，但还在每天捡垃圾，住烂尾楼。小A就是持续稳定的在出租屋睡觉，上网，偶尔换个地方，去别的城市，然后继续出租屋睡觉，上网<br/>
<br/>
第三 他不开充电，不接受打赏。这点是我最佩服的，毕竟这类UP主十个有九个都想快点赚钱，小A竟然把来钱最快的途径给掐了，顶多接个被子按摩仪之类B站的常规广告<br/>
<br/>
第四 小A不高线下聚会，不鼓励别人去找他，偶尔被人发现都比较排斥，我印象里也没搞啥粉丝群。反正就是独狼性格，没想着要搞啥粉丝经济。<br/>
<br/>
第五 小A还很正能量，这类卖惨生活困顿的UP主偶尔还会批评一下自己生活怎么变成这样的，社会福利、基础保障、失业率之类的事儿，聊着聊着就往键政的路子上走了，但小A几乎不碰这些，偶尔提到社会话题都是非常正能量，支持国家各项政策，是个很乖的人。<br/>
<br/>
人就靠自己的朴实，亲和力，接地气，不装逼，收获了上百万的粉丝。人也没想开直播带货，有这么多粉丝也没签MCN，没想着赚大钱，发的视频依然和两年前没啥区别，依然过着空调不咋制热的出租屋里的平淡生活。<br/>
然后一个月靠视频跟几个小广告的收益，也能过活了。<br/>
<br/>
不给党和国家添麻烦，自给自足。<br/>
<br/>
但是，就这么一个不喷东大，不爆老登金币，不收打赏，一心纪录自己睡觉和上网生活的UP主，也要被东大的铁拳看不惯，给砸了。<br/>
就这么人畜无害的一个家伙，都不给他一口饭吃。<br/>
<br/>
我只能说，有时候把人逼急了，可真指不定闹出什么事来。

<div class="detail_wbtext_4CRf9">
	在看一个韩国综艺《侦探们的营业秘密》<br/>
2020年法律修改后侦探在韩国成为了正式受认可的职业之一。来找侦探的案件80%是出轨，剩下20%是企业犯罪和寻人。为了拍出这个综艺，剧组拜访了韩国几乎每一位侦探，听到了很多惊人的案件。每一集的上半部分是律师、侦探和主持人们一起在演播厅观看真人真事翻拍的故事并发表看法，下半部分是跟着实际工作的侦探全程跟拍（侦探vlog了就是说）<br/>
看到其中一位主持人在油管说这个综艺心太累了，因为故事太逆天，拍摄完总是所有人身心俱疲。激起了我的好奇心！开始看！（感恩b站有生肉）<br/>
有趣的TMI: 侦探们年纪有大有小，其中不乏👮‍♂️出身，之前缉毒的重案组的都有，也有很多摩托车手（跟踪一个人的时候一辆车+一台以上伪装外卖/快递的摩托车），也有打架能手（跆拳道四段，做过保镖），也有女性（很多场景单身男人一人潜入不太自然，看情况伪装成情侣或者女性潜入）<br/>
偶尔也有六十左右退休的企业高管来追梦，也能加入，这些人穿上西装一坐下那就是个高管样没有任何人会怀疑，非常合适。侦探的工作大部分是脚踏实地的低效率工作（打上千个电话，走访上千个人只为一个线索），也有E人侦探人脉广，坐着不动打几十个电话就能找到的（据说买了无数个保险，开了无数张卡，别人找他推销他就买，这一切都是卖人情，帮了这个忙一般问点事没人会拒绝，于是光靠电话就能找到目标）<br/>
第一集大概内容：一位40岁出头家庭主妇抱着婴儿来侦探事务所求救，她在老公的车里看到写给情人的贺卡鲜花和礼物，怀疑老公出轨。最近小女儿出生，老公努力跳槽到了高薪大企，从那时起老公像变了个人一样，原本关系很好，现在情绪激动，拒绝任何肢体接触，老婆只要追问老公就说不然我辞职行不行！老婆便不敢多说什么。<br/>
侦探跟踪老公，看到他开车进小巷，从破旧的楼房接上情人一起去酒吧喝酒，之后又去酒店，还非常谨慎地让情人先下车，他绕一圈再进，出来的时候也是制造时间差，侦探没有拍到两人一起进出酒店的照片，当天作罢。第二天是夫妇的结婚纪念日，料他不敢见情人，侦探休息了一天，第二天收到委托人电话，说昨晚（结婚纪念日）老公自杀死亡。<br/>
侦探赶往葬礼得知昨晚老公捧着蛋糕鲜花回家，说自己记得结婚纪念日的，一家四口难得开开心心吃完饭，等两个孩子睡了说好夫妻小酌一杯，老婆准备下酒菜时老公接了个电话当场跳楼自杀，老婆崩溃。<br/>
葬礼中和老公最后一次通话的上司说老公聚餐中一个人消失，担心所以打了电话，得知老公回家了就挂了。老公的手机通话录音都删了个干净，只剩这最后一通的录音，录音内容证实了上司的说法。老公似乎有抑郁症，同事们都知道，只有老婆不知道，她非常内疚。<br/>
侦探觉得蹊跷，前天还和情人进酒店怎么突然跳了，这时侦探观察到上司走路外八很眼熟，重看酒吧和酒店外的录像，看到了隔着几步在老公和情人附近走路的上司。这三人之间是什么关系？<br/>
跟踪上司得知原来情人是上司的情人，那么老公（死者）为什么一直和他俩在一起？去问公司同事大家心照不宣一问三不知，只有一个人偷偷给侦探塞了已经离职的前同事联系方式，通过前同事得知上司是裙带关系，在公司无法无天，霸凌下属，老公作为新员工被盯上了，为了生存只得讨好。上司因为自己出轨被老婆抓过，无时不刻带着老公当人肉不在场证明，每次开房都说在加班，上司的老婆打来电话就叫隔壁房等待的老公（死者）接电话作证，在公司也是百般刁难当众侮辱，每天还要检查手机，删除所有证据（包括通话录音）。老公患了恐慌，又不敢辞职，因为上司扬言他要是离职会在业内散播消息在没有公司会收他，老公就这样每天靠吃药硬撑。<br/>
为了收集证据，侦探作为保洁进公司，和保安打听到这个公司就是这样的，人心惶惶，和员工问不出什么，侦探想到老公多么老实一个人，登上了他的云盘，果然一切通话内容和聊天记录早已备份。老婆拿着证据去找公司法务部，法务拒不承认这些记录和老公跳楼有直接关系，公司不会赔偿。老婆听说当天晚上老公被上司叫到了外面，但发生了什么没有任何证据也没人作证，希望公司调查，公司拒绝。<br/>
于是侦探又开始走遍当天聚餐的酒吧街寻找目击证人，最后找到每天晚上停车到聚餐场所对面停车位的补习班面包车，要来了黑匣子录像。根据录像，当天老公在聚餐中说今天是自己的结婚纪念日一定要回家，被上司叫到店外空地暴打了一顿（专挑看不出来的地方打），老公被打后没有回到聚餐，直接回家见家人，之后又接到上司的电话，崩溃跳楼。<br/>
通话内容乍一听很平常。“李组长，你在哪？”，“我在家。”“家？我们聚餐聚到一半你回家了？刚刚不是都和我说好了吗？”“我…我在陪家人…”“啊～这样，那应该的，你是家长嘛，当然家人最重要了，那明天我们公司见呗”<br/>
老婆这才得知老公每次情绪激动地说“那我就辞职行不行”是求救信号，她很内疚没有听出来，之后拿着这些证据进行长期战，向公司索要道歉和赔偿，公司拒不承认赶她走，最后没有办法在网上暴露，公司受舆论压迫不得已进行赔偿。上司被家人（公司高层）和公司抛弃，那些长期受他霸凌、性骚扰的员工们集体起诉，情人也抛弃了被老婆离婚后屁都不是的他。<br/>
据说这是韩国侦探业内很有名的一个案子，刚看完第一集上半部分我已经累了。

	昨晚跟老家那边通话，一个老街坊说起现在年轻人工作不好找，到处裁员辞退的。我说是，幸亏咱们身边这几个还都没遇上，娃儿些运气好。<br/>
“对吧你看你也这么说！”她大喊，“所以我就坚决不准他辞职！不得准！”原来她娃儿想辞职，说实在忍不了了，虽然已经干了八九年。<br/>
我问具体啥原因辞职，她说不晓得，不管啥原因都不准辞职，给他说了不准。我说你问问到底为啥，万一有什么严重的情况呢？她说没有，真要有严重的情况他不会不说，他就说工作太老火忍不下去了。“吃不得一滴点苦！”她苦笑。<br/>
我还是觉得不妥但也没法再多话。<br/>
“都在劝，大家都在帮到劝，帮到开导，你也说了他运气好劝他要珍惜不要乱整，啥辞职都来了哟……”她感谢我。我听了脑袋嗡的一下，马上更正说“我没劝他别辞职，我只说他没遭遇裁员很幸运”，我控制不住情绪口气变得生硬。她觉得诧异，说明明刚才我说了“运气好”，言下之意就是要珍惜别胡来呀？再说又不是我一个人这么说，大家都这么说。<br/>
我问大家都谁？她说多了，说了好几个老头儿老太太，他们都觉得不能同意娃儿辞职，说他根本不知道责任，你倒是辞职了，你两个老的怎么办？再难都要挺住，都这么过来的。“你妈也说了，还给我发了好几个视频，说现在工作难找，能干下去就得干下去…”<br/>
我回去一问果然，我妈真的给她发了视频，还鼓励她给孩子看看，参考参考，多知道点儿大环境的信息总是有帮助的。再看那几个视频，也的确不是胡说八道，但无非同样的内容互相抄抄，一样的意思由好几个人说，显得人声鼎沸。我求我妈再也不要发她了，对她娃儿要辞职的事别给任何建议，“万一你们扼杀了一个创业天才呢？万一。”我妈觉得我危言耸听但想想也怕万一，就答应了。<br/>
什么具体情况都不了解不想了解，就不准坚决不准，就敢给建议给教导，就敢干预别人的人生，不管年轻人怎么想，我先麻爪了。
