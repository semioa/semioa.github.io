	想到在有微信之前，人好像是可以不用一直在线的。那时候人和人之间的联系很贵，也很延时，一条短信要花上几毛钱，打电话不仅要话费，也总是显得太隆重，因此“找不到人”是一种常态。更不用说qq的时代，不登陆就联系不上，头像会变成“下线”的灰色。而即使上线，也可以隐身，也可以在别人找你的时候说“不是本人”――这么好用的理由，却再也用不上了。<br/>
如今微信几乎把每个人都捆绑在了线上，需要时刻关注，时刻回应，一个人不回消息会被认为毫无社交道德，回的太慢也是冷漠怠慢，甚至很多关系里“秒回”才是一种爱意的呈现。<br/>
也许人的脑子，自我空间，生活，思考，呼吸，都是被这种强制性的“在线”偷走的。

	【2】两位经济学家刚刚发表了一篇数学证明，表明人工智能将摧毁经济。<br/>
<br/>
不是可能。不是也许。就是会――如果什么都不改变。<br/>
<br/>
这篇论文名为《人工智能裁员陷阱》。发表于2026年3月2日。宾夕法尼亚大学沃顿商学院。波士顿大学。同行评审。数学建模。<br/>
结论只有一句话。<br/>
“在极限情况下，企业通过自动化实现无限生产力，同时需求降为零。”<br/>
<br/>
一个生产一切的经济体。却无人购买。<br/>
以下是通往此境的路径。<br/>
一家公司解雇500名工人，用人工智能取代他们。一家竞争对手解雇700人以跟上步伐。另一家解雇1000人。每家公司都在理性行事。每家公司都在正确遵循激励机制。而每家公司都在为自己挖一个陷阱。<br/>
<br/>
因为那些被解雇的工人也是顾客。<br/>
当他们失业速度超过经济吸收能力时，他们停止消费。消费者需求下降。公司通过削减成本应对――这意味着自动化更多工人――这意味着更少消费――这意味着需求进一步下降――这意味着更多自动化。<br/>
<br/>
这个循环没有自然出口。<br/>
研究人员测试了所有提议的解决方案。全民基本收入。资本所得税。工人股权参与。技能提升计划。企业协调协议。<br/>
模型中每一个都失败了。<br/>
唯一有效的干预措施：皮古维自动化税――每次公司用人造智能取代人类时，按任务征收费用，迫使他们在扣动扳机前将自己破坏的需求计入价格。<br/>
<br/>
没有政府实施过这个。没有主要经济体在认真讨论它。<br/>
与此同时，数字已经跟踪上这条曲线。2025年10万科技工作者被裁。2026年初再裁9.2万。杰克・多西解雇了Block一半员工，并公开表示：“在未来一年内，大多数公司将得出同样的结论。”<br/>
没有人做错任何事。公司们完美遵循了它们的激励机制。这恰恰就是问题所在。<br/>
<br/>
理性行为。以规模化。同时发生。没有机制来阻止它。<br/>
两位经济学家构建了数学模型。数学模型指向一个地方。<br/>
<br/>
来源：Falk &amp; Tsoukalas ・ 沃顿商学院 + 波士顿大学

一旦它做错了事，你开始骂它，那它在接下来的对话里就会以更高的频率犯错和出现幻觉，越做越乱。<br/>

<br/>
这时候唯一正确的做法是认赔杀出，不要继续这个对话（当然更不要结束前还毫无意义地骂它一顿），而是果断新开对话 session。<br/>
