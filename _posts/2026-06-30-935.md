<div class="detail_wbtext_4CRf9">
	看到最近古二这事，我多年北漂编剧生活已结痂的ptsd属实复活了一下。<br/>
但目前我已康复五六成，明白沉溺于情绪很难改善编剧处境，所以借着近期《繁花》的争议，跟所有关心国产影视的网友、以及愿意一起努力的业内同仁们聊一聊，编剧的处境到底怎么了？<br/>
这个话题实在太大了，语境幽深，每每说起长叹一声，好难讲清。我今天从两个方面来试着跟大家聊明白。<br/>
一个是【现行“编剧”定义太粗泛，装不下复杂协作的现实】<br/>
首先讲讲【编剧】和【作家】之间的区别。<br/>
大众朴素的认知，容易将编剧和作家混为一谈，实则猫和猫头鹰根本不是一个物种。作家相比之下可以“任性”，更像大众印象中的“艺术家”；编剧要兼顾巨大的商业回报，一半文艺一半工匠，工匠的那部分像建筑师、工程师、程序员架构师。<br/>
（毕竟，作家写出的小说本身就是成品，直接面对受众，打垂类，十万销量已是傲人表现；而编剧写出的剧本是半成品，是剧组的生产图纸，是数千万乃至数亿资金如何花的计划书，有一千万人看都算扑街，需要巨大路人盘。）<br/>
作家原则上是不允许任何一个字代笔的，但编剧有团队协作分工是一种健康正常的局面，和独立编剧一样健康正常。<br/>
编剧有团队，并不等于代笔、压榨、不敬业、欺世盗名，这是对编剧工作性质的误解。很多年轻编剧都是从成熟编剧的团队中成长锻炼、自立门户的，良心好的前辈提携晚辈，良心差点的压榨晚辈，也可能一边提携一边压榨同时进行――好赖并存，但这是目前编剧最主要的成长路径之一。<br/>
问题在于，共同创作的编剧们是怎么分工？大家应该拥有怎样的title？谁来决定title怎么给？<br/>
再说说【编剧团队】和【导演团队】的不同。<br/>
导演也不是一个人在工作，大项目的导演组一般有十几人到几十人：总导演带领着分组导演、执行导演、演员副导演、现场副导演、后期导演、视效导演、场记、导演助理等，协同工作，分工各异――假如导演组成员全部统一都叫“导演”，那必定也要为了名利而扯头花了。<br/>
另外，编剧的工作不像导演每天在现场众目睽睽的，剧本的诞生更“黑箱”，工作过程漫长繁杂不可见，所以很容易闹出罗生门，每个人都觉得自己贡献更大。<br/>
编剧团队的黑箱工作状态，非常非常非常考！验！人！性！容易陷入抢功劳扯皮的局面。<br/>
我既见过大编剧欺压小编剧，也见过小编剧作妖诬陷提携自己的大编剧――而且你真的很难说哪种情况更多。现实中的弱者是网络舆论中的强势方，现实中的强者又是网络舆论的弱势方……忠奸难辨。<br/>
这就是为什么每逢署名争议，出来表态的业内不多，并不是所有都那么人情世故怕得罪人，也是确实不敢轻易下判断，不少人都因为帮别人维权而被打过脸，而且也深知编剧的处境，是系统性的失权，结构性的困境，不一定能三言两语分出清晰简单的善恶对错。（再次强调我是在说大部分情况，而不是在聊繁花事件)只能说，小编剧的处境更艰难，编剧们不分大小也更倾向于同情小编剧，毕竟都是牛马，会同情更穷更惨的牛马，物伤其类。<br/>
平心而论，一部剧把所有参与过剧本的人都署名为“编剧”，对真正起关键作用的那一两个核心编剧也是不公平的。所以如果能在片尾详细增加“故事研发”、“史料顾问”、“对白润色”、“方言指导”等等带具体职责的署名栏，和“编剧”并列出现，既保留创作链的清晰度，也不稀释“编剧”这一职位对成片结构和叙事风格总负责的含义。<br/>
当然，如果参与了剧本的某个咔拉米对主线、人物塑造都发挥了很重要的作用，作为总编剧，帮人家跟片方争取一个片头编剧的title也是美事一桩，真的没必要那么吝啬。<br/>
说一千道一万，方法再好，也是人执行的。<br/>
整个行业对剧本漠视轻视至此，谁会给你分这么细呢，连总编剧地位都不行，署名都不一定保得住，何况其他编剧组成员。<br/>
到底什么人贡献足够大？到底谁配得上署“编剧”？这又是一笔糊涂账。这就涉及到今天要说的第二点，也是编剧更大的一个痛点――<br/>
【定稿权彻底旁落，编剧沦为各路人马的人肉打字机】<br/>
从古二的控诉，大家也不难看出，剧本的裁判是导演王家卫，他说好那就是好，他说不好就必须改。当然了，那毕竟是王家卫，《繁花》也是个极为特殊的项目，方方面面都特殊。<br/>
但在绝大部分项目里，编剧就是一个听起来尊贵，实则没有话语权的位置，被一大群不专业的人指挥写剧本，然后剧播出时背所有黑锅。已经不是戴着镣铐跳舞的问题了，严重的情况下像边被凌迟边跳舞。<br/>
大部分编剧之间扯皮事件，给我的感觉都像看到雌竞一样难受，仿佛那种大房和偏房、外室之间的纯恨一样。郎君分明觉得我最好，为何你这个小蹄子却登堂入室……<br/>
正妻比外室强势，但也不能说封建社会女性地位是ok的，就算有生了儿子的甚至跋扈起来，也改变不了整体受压迫的事实。编剧互撕，无非是因为对方也只是个下九流，谁也不敢剑指真正的话事人决策者。<br/>
抛开繁花事件的具体当事人不谈，没叠加出品人、投资方身份buff的大编剧也仅仅是比小编剧强，在整个工业链条里是非常弱势的位置，弱势、委屈到观众其实想象不到的程度――原因太多了，只能单独开一篇来聊。<br/>
总结来说呢，就是影视行业创作生态存在问题，编剧处境糟糕折射出的是全行业对创意的漠视。<br/>
如果编剧处境问题迟迟不解决，眼下已经很严重的人才断代、观众流失、资金撤离，都会越来越严重。但我这一篇微博其实没有什么结论的，我要是有本事解决这些问题，我也不是个只能在网上bb的咔拉米了。<br/>
不过聊聊也是好的，你不说，我不说，都安静如鸡，不就是在喧嚣的沉默中毁灭吗。

	【6】运动与血糖波动<br/>
<br/>
有少部分人知道，运动可以燃烧葡萄糖，改善胰岛素功能，运动后血糖水平通常会下降。由于很多人平时不运动，葡萄糖老无所依，身体拼命分泌胰岛素来降低葡萄糖，但细胞快被撑死了，决定把门关上，不要葡萄糖了，因为主人不需要燃烧葡萄糖，胰腺最后也累死了，久而久之就是2型糖尿病。<br/>
<br/>
现在，有了连续血糖监测仪，如果你佩戴连续血糖监测仪，同时没有代谢综合征问题，又喜欢运动，可以监测每天24小时的血糖变化。这时你会发现，你在运动时，尤其是比较剧烈的有氧或无氧运动后，血糖会上升。如果正常值是90，运动后可能飙升到150-200。<br/>
<br/>
如果你是个血糖正常没有代谢问题的人，完全不用为此担心，因为这是我们祖先留给我们保护自己的方式。<br/>
<br/>
葡萄糖是给细胞提供能量用的，这是你平常有劲儿的原因之一。尤其是你在做各种阻力训练或快跑过程中，需要大量葡萄糖。我们的祖先每天早上从山洞里出来，要出去打猎，每次出门都意味着有三种可能：满载而归、空手而归、搭上小命。<br/>
<br/>
你可以想像我们祖先可比现在有力量，但他们需要团队合作才能制伏一头狮子或老虎。当他们与猛兽搏斗时，肾上腺素，也就是应激激素会激增，当应激激素激增，实际上是给肝脏点烽火，肝脏看到冒烟就会往血液里释放肝醣（一种储存在肝脏中的稠密且稳定的形式的糖原），让血糖水平迅速升高，以便为肌肉提供能量。<br/>
<br/>
实际上，当你每天早上一睁眼，肝脏就开始分泌肝醣，你洗脸的时候都需要葡萄糖。而且，身体优先使用这种肝醣，只有用完了（比如剧烈运动），才会动用血液里的葡萄糖――你试想一下，如果你不运动，葡萄糖一直在血液里，逼着胰岛素不断分泌来把血糖压下来（送进细胞），细胞要不了那么多葡萄糖，要么把它变成脂肪，要么又送回肝脏，肝脏要么储存成糖原，再释放到血液里；要么变成内脏脂肪（脂肪肝和大肚子）。这就是肥胖的来历。<br/>
<br/>
分泌肝醣不及时的祖先都被猛兽吃了，剩下的祖先们这个功能进化得更加完善。你得对得起祖先付出的一切。<br/>
<br/>
今天，当你做一些剧烈运动，你的身体会认为你遇到了猛兽，所以会配合你分泌大量葡萄糖，怕你被猛兽吃掉。因为身体会认为你处于“战斗或逃跑”状态，要么你满山追老虎，要么被老虎追的满南京路跑。不管是什么状态，你都需要葡萄糖。另外，运动会产生乳酸，乳酸会转化成葡萄糖为肌肉提供能量，这是一种内循环。<br/>
<br/>
在这个过程中，由于肝脏瞬间释放了大量肝醣，但是不参与运动的细胞并不需要葡萄糖（比如你的脸蛋子），于是出现暂时性胰岛素抵抗，这样葡萄糖会进入运动肌肉的细胞，让你四肢充满力量。<br/>
<br/>
武松当年打虎，如果佩戴连续血糖监测仪，可能血糖会飙升到270（15mmol/L）。这就是为什么打完老虎，武松瘫软在地上，一点力气都没有，原文是“原来使尽了气力，手脚都苏软了。”然后他歇了一会儿，之后走了半里路，此时他的肾上腺素、血糖迅速降下来，心率也平稳到90BPM钟左右，之前运动肌肉（四肢）处于能量暂空状态，实际上他非常疲惫。这时看到猎户来了，以为又来了老虎，说道：“阿呀！我今番罢了！”因为这时候潘金莲都能把他掐死，更别说两只老虎。<br/>
<br/>
神医安道全看到武松打虎这一段，点了点头：施耐庵还是懂点内分泌学的。<br/>
<br/>
如果你希望运动期间保持血糖稳定，运动前摄入一点碳水化合物。<br/>
<br/>
有2型糖尿病，要不要剧烈运动呢？要得！当然，这要看你的实际身体情况，最好循序渐进，你不可能像你当初正常那样剧烈到不管不顾。总体来说，剧烈的有氧或无氧运动会提高胰岛素敏感性。<br/>
<br/>
2型糖尿病的人可以先从快走、骑行、游泳、爬楼梯、跳舞等中强度运动开始。当然，这些你可能都不喜欢，不然也不会2型糖尿病。但只要坚持下来，对恢复胰岛素敏感性是有帮助的。而且，每周至少3次力量训练――它对糖尿病管理、糖前和出现胰岛素抵抗的人非常有效。

泰诺品牌旗下药物有四种，分别为：<br/>
<br/>
1. 泰诺-酚麻美敏片，这个药是用于成人感冒，大家平时说的「泰诺」一般指的就是治疗感冒的这个泰诺。每片酚麻美敏片含有：<br/>
<br/>
-对乙酰氨基酚（325mg，解热镇痛药）：缓解发热和头痛等症状；<br/>
-右美沙芬（15mg，中枢镇咳药）：缓解咳嗽症状；<br/>
-氯苯那敏（2mg，组胺拮抗药）：缓解打喷嚏和流鼻涕症状<br/>
-伪麻黄碱（30mg，减充血药）：缓解鼻塞。<br/>
<br/>
2. 泰诺林-对乙酰氨基酚缓释片，用于成人的解热镇痛，主要成分就是对乙酰氨基酚。<br/>
<br/>
每片泰诺林含有650mg对乙酰氨基酚，剂型为双层片，由速释层和缓释层两部分组成，口服后速释层迅速溶出起效，而缓释层缓慢释放持续8小时作用。注意泰诺林片用于成人和12岁以上儿童，每次1片，每8小时1次，24小时内服用不超过3次。<br/>
<br/>
3. 泰诺安-氨酚拉明片，用于成人的夜间镇痛。<br/>
<br/>
相对于泰诺家族的其他药，泰诺安的知名度低一些。它也是一个复方制剂，每片含有500mg对乙酰氨基酚和25mg盐酸苯海拉明。苯海拉明是一种抗组胺药，但是有嗜睡的副作用，所以用药期间不能开车或者从事一些有风险的工作。<br/>
<br/>
4. 泰诺林-对乙酰氨基酚混悬滴剂/口服混悬液，用于儿童的解热镇痛，主要成分和2一样，也是对乙酰氨基酚。<br/>
<br/>
儿童泰诺林有两种剂型，一种是15ml的混悬滴剂，另外一种是100ml的口服混悬液，这个跟美林一样，美林也分为小美林（布洛芬混悬滴剂）和大美林（布洛芬混悬液）。小泰诺林用于3个月以上儿童，大泰诺林用于1岁以上，其实成分是一样的，主要是方便不同年龄段的婴幼儿服用，看瓶身上画的宝宝图可以分辨出来。<br/>
<br/>
上面几种泰诺，只有泰诺-酚麻美敏片是不能在线上购买的，而且线下药店也需要出示身份证购买，原因在于其中含有的伪麻黄碱成分，该成分在之前管理没那么严格时会被用于非法目的，其他几种含有该成分的复方感冒药（如新康泰克-氨麻美敏片）目前也是相同的管理方式。<br/>
<br/>
另外强调一点，很多感冒患者并不是所有症状都出现，比如有些人只有发烧和头痛，那么只用对乙酰氨基酚（或者布洛芬）就足够了，并不需要复方感冒药中的其他成分。<br/>
<br/>
这一点在儿童方面尤为重要，为了防止感冒药副作用对婴幼儿的伤害，FDA已经全面禁止销售2岁以下儿童复方感冒药，欧洲和美国也不推荐6岁以下儿童使用复方感冒药。<br/>
<br/>
希望以上内容能让大家区分不同的泰诺。
</p>

<p>
	【8】自闭症的成因要公布了吗？科学家们用数据回答！川普声称，他的政府会在今天公布“自闭症的成因”。在此关键时刻，国际科学权威杂志《自然》刊登了一篇综述文章，汇总了最新的科学证据。核心信息是：自闭症的确切成因至今没有定论！<br/>
<br/>
在欧洲糖尿病研究协会年会上，一项涵盖20个国家48项研究、900多万次妊娠的数据分析显示：妊娠期糖尿病母亲所生的孩子，未来罹患ADHD（注意缺陷多动障碍）风险增加36%，自闭症风险增加56%，这些孩子的智商（IQ）得分更低。<br/>
<br/>
类似结论也出现在国人喜爱的国际医学顶级期刊《柳叶刀 糖尿病与内分泌学》刊登的另一项超大规模荟萃分析结果：5600万对母婴数据显示，孕期任何类型的糖尿病（I型、II型或妊娠期糖尿病）都与孩子未来罹患ADHD和自闭症风险上升相关。<br/>
<br/>
科学家们在解读时非常谨慎：荟萃分析或观察性研究并不能确定“因果关系”！目前没有任何研究能确认：妊娠期糖尿病会直接导致自闭症。现有证据只显示相关性，并不能证明“因果”。<br/>
<br/>
（1）基因是主导因素：新加坡国立大学研究者李玲君（音译）指出：遗传因素对ADHD和自闭症的影响最大。这一点有最广泛的共识！而妊娠期糖尿病、叶酸缺乏、空气污染等环境因素只是次要“推手”，属于小到中等程度的影响。<br/>
<br/>
（2）多重因素交织：墨尔本大学的产科医生Alex Polyakov强调：“在实践中，往往是遗传易感性和环境因素相互作用”。<br/>
<br/>
换句话说：糖尿病可能是风险因素，但它背后很可能是基因易感性+环境触发共同作用的结果。<br/>
<br/>
所以，即使川普公布了所谓的自闭症成因，无论是疫苗还是孕期使用泰诺，科学界大概率都不会承认，因为截止到目前，并无令人信服的成因研究！
</p>
